

# 测评结果页结构 & UI文案优化方案

## 一、当前实现 vs 目标规范对比

| 模块 | 目标规范 | 当前实现 | 优化点 |
|------|----------|----------|--------|
| **模块1：状态概览** | 标题"你的情绪状态概览"，副标题"基于你刚完成的测评结果生成" | 标题"第一层：状态筛查结果" | 需改为更友好的标题 |
| **模块2：反应模式** | 标题"你现在最主要的情绪反应模式" | "第二层：反应模式识别" | 需优化标题文案 |
| **模块3：卡点聚焦** | 标题"你目前最需要优先修复的是" + 单点聚焦说明 | 已有类似结构 | 需要优化说明文案风格 |
| **模块4：今日第一步** | 标题"今天你可以先从这一步开始" | 已实现 | 文案微调 |
| **模块5：行动按钮区** | 主按钮+副说明+次按钮 | 已实现 | 需优化层次 |
| **合规声明** | 页面底部固定小字灰色 | 已有 | 微调文案 |

## 二、详细优化清单

### 2.1 模块1：顶部状态概览

**当前**：
```text
标题：第一层：状态筛查结果
副标题：基于你的作答生成的三大指数
```

**优化为**：
```text
标题：你的情绪状态概览
副标题：基于你刚完成的测评结果生成
Badge：需要关注 / 适度留意 / 状态良好
```

**新增**：
- 移除"第一层"等技术性标签（用户不需要知道三层模型）
- 保持三个指数的横向展示
- 保留底部解释小字

### 2.2 模块2：情绪反应模式

**当前**：
```text
小字标签：第二层：反应模式识别
标签：主要模式
```

**优化为**：
```text
标题：你现在最主要的情绪反应模式
大标签卡片：🎯 【高度紧绷型】
一句话：你对自己要求很高，但几乎不给自己真正放松的空间。
副类型标签：次要模式：情绪压抑型
```

**调整**：
- 移除"第二层"技术性标签
- 大标签卡片更突出展示主类型
- 折叠区域保持：典型表现 + 内在机制

### 2.3 模块3：行动阻滞点（你现在最卡的地方）

**当前**：
```text
小字标签：第三层：行动阻滞点
标题：你目前最需要优先修复的是
```

**优化为**：
```text
标题：你现在最卡的地方
副标题：你目前最需要优先修复的是
聚焦卡片：【大脑持续警戒状态】
说明：更具体的机制解释
```

**调整**：
- 移除"第三层"技术性标签
- 卡片内的说明文案需更具体、更指向行动

### 2.4 模块4：今日第一步建议

**当前标题**：今天你可以先从这一步开始

**保持**：标题和结构已符合规范

**微调**：行动卡片内的文案格式优化

### 2.5 模块5：行动按钮区

**当前**：
```text
统一承接区卡片
主按钮：开始我的AI陪伴对话
次按钮：查看完整成长支持路径
```

**优化为**：
```text
主按钮（强）：🟠 开始我的 AI 陪伴对话
副说明：根据你的状态，我会陪你一步步调整节奏
次按钮（弱）：查看完整成长支持路径 ＞
```

**调整**：
- 主按钮增加副说明文字
- 优化按钮层次感

### 2.6 合规声明

**当前**：
```text
本测评为情绪状态与成长觉察工具，不构成任何医学诊断或治疗建议。
若你感到持续严重不适，请及时联系专业心理支持机构。
```

**优化为**：
```text
本测评为情绪状态与成长觉察工具，不构成任何医疗诊断。
如你感到持续严重不适，请及时寻求专业心理支持。
```
