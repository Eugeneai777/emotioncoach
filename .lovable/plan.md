
# 为 AI教练解说按钮添加价值提示

## 目标
在按钮周围添加价值点列表，让用户一眼就知道点击后能获得什么，提升点击率。

## 修改文件
`src/components/wealth-block/AssessmentVoiceCoach.tsx`

## 设计方案

在按钮下方添加 3 个简洁的价值标签，横向排列：

- "深度解读报告" — 让用户知道教练会针对测评结果做详细分析
- "个性化建议" — 强调是针对个人情况的定制方案
- "实时语音互动" — 突出可以对话提问的独特价值

### 视觉设计
- 使用小型圆角标签（pill），带图标（如 FileText、Target、MessageCircle）
- 颜色与按钮呼应（红/玫瑰色调的浅底色）
- 字号 text-[10px]，不喧宾夺主但清晰可读
- 使用 framer-motion 做入场动画（依次淡入）

### 布局
按钮 + 标签整体保持居中，标签在按钮正下方，间距紧凑（mt-3）。

## 技术细节
- 仅修改 `AssessmentVoiceCoach.tsx`
- 在按钮 div 内、button 元素下方添加价值标签行
- 不改变组件 props 和外部接口
