

## 优化"短期目标"区域：增强训练营迫切性与价值传达

### 问题分析

当前"7天后 85+"和"毕业 90+"两个目标框只显示数字，缺少对"达到这个分数意味着什么"的解释，用户看不懂价值。

### 改进方案

**文件：`src/components/wealth-block/AwakeningJourneyPreview.tsx`**

#### 1. 为"7天后"目标添加价值说明

在"85+ 短期目标"下方增加一行具体描述，让用户知道达到这个分数意味着什么：

- 如当前 awakeningStart < 40：显示"从觉醒起步跨入初步觉醒"
- 如当前 40-59：显示"突破初步觉醒，进入稳步觉醒"
- 如当前 60-79：显示"迈入高度觉醒区间"
- 动态计算，基于 day7Target 对应的觉醒区间名称

#### 2. 为"毕业"目标添加区间描述

- 显示"财富能量畅通"（对应 80-100 高度觉醒区间的描述）

#### 3. 增强视觉迫切性

- "7天后"框：将 border 从 `dashed` 改为 `solid`，加深颜色，增加微动画（脉冲光晕）
- 在起点和目标之间添加一个上升箭头动画，暗示增长趋势
- "短期目标"文字改为更具体的描述文案，如"你将到达"

#### 4. 强化训练营介绍文案

- 在训练营概览框中增加一句"痛点呼应"：根据用户的 dominantPoor 类型，显示"你的[嘴穷/手穷/眼穷/心穷]模式正在消耗你的财富能量"
- 将"7天·每天15分钟"标签加大为更醒目的样式

### 具体代码修改

| 区域 | 当前 | 改为 |
|------|------|------|
| 7天目标副标题 | "短期目标" | 动态文案如"突破到稳步觉醒" |
| 毕业目标副标题 | "高觉醒" | "财富能量畅通" |
| 7天框边框 | `border-dashed border-emerald-400` | `border-solid border-emerald-400` + 脉冲动画 |
| 起点→目标之间 | 无 | 添加 `↑` 上升箭头动画 |
| 训练营概览 | 通用文案 | 增加基于 dominantPoor 的痛点呼应句 |
