## 海报 v3 优化方案

在 `/tmp/make_posters.py` 基础上修改并重新生成两张海报（输出 `poster-A-pay_v3.png` / `poster-B-wechat_v3.png`），保留原版作为对照。

### 1. 文案修改（两张海报同步）
- 「回应模板」→「回应练习」（A 版交付列表第 3 项副标）
- 「妈妈，我想跟你聊聊」→「爸爸妈妈，我想跟你聊聊」（主标语副文）
- 所有「倾听教练」→「心理教练」（页眉 chip、人物卡身份标签）
- 删除人物卡底部「持证心理咨询」一行

### 2. 人物卡（静静 Rebecca）布局优化
当前：人像 + 底部「静静 Rebecca / 倾听教练 · 持证心理咨询」双行。
调整为：
- 仅保留「静静 Rebecca」主名 + 「心理教练」副标（单行排布，居中对齐）
- 名字字号略增大，副标用 muted 绿色描边气泡
- 底部白色信息条高度收紧，给人像留更多视觉空间

### 3. 人像处理（核心改动）
原图三人：左侧站立女生 + 右侧背影女生 + 中间长椅上女主角。
需求：只保留中间长椅上的女孩，去除左右两人。

技术方案：
- 使用 AI 图像编辑：调用 `imagegen--edit_image`，输入原始人像，prompt 指定"移除画面中除中间长椅上女孩之外的所有人物，保留书架/长椅/木质背景的自然延续"，aspect_ratio `3:4`
- 输出至 `/tmp/coach_clean.png`，再由 Pillow 脚本贴入人物卡圆角遮罩内
- 若生成结果不理想（边缘穿帮/人物变形），降级为局部裁剪：将人像裁剪框收紧到中间女孩 + 长椅，左右各裁掉约 25%，并在人物卡内重新居中

### 4. 输出与 QA
- 生成 `/mnt/documents/poster-A-pay_v3.png` 与 `poster-B-wechat_v3.png`（1080×2400）
- 生成后用 `code--view` 检查：① 文案全部替换到位 ② 人像无穿帮第二人 ③ 「心理教练」字号与气泡协调 ④ 整体留白与原版一致
- 交付 `<presentation-artifact>` 标签供预览下载

### 不改动
- 配色、字体、价格卡、二维码区域、底部 slogan 均保持 v2 版本不变
