

## 移除"体验包内容"分类

### 问题分析

截图中显示的"体验包内容"部分会误导用户：
- 三个等级的体验包内容完全相同（尝鲜会员50点、情绪健康测评1次、SCL-90心理测评1次、财富卡点测评1次）
- 用户看到后会觉得"既然内容一样，为什么要选更贵的等级？"
- 真正的差异在于"体验包分发权数量"（100/500/1000份），这已经在基础信息中展示了

### 修改方案

从 `src/config/productComparison.ts` 中移除"体验包内容"分类的4行数据。

### 修改内容

| 文件 | 操作 |
|:-----|:-----|
| `src/config/productComparison.ts` | 删除第89-93行的体验包内容数据 |

### 技术细节

移除以下代码：

```typescript
// 体验包内容（新类别）- 删除这4行
{ name: "尝鲜会员", category: "体验包内容", l1: "50点", l2: "50点", l3: "50点" },
{ name: "情绪健康测评", category: "体验包内容", l1: "1次", l2: "1次", l3: "1次" },
{ name: "SCL-90心理测评", category: "体验包内容", l1: "1次", l2: "1次", l3: "1次" },
{ name: "财富卡点测评", category: "体验包内容", l1: "1次", l2: "1次", l3: "1次" },
```

同时，可选择性地从类型定义中移除 `'体验包内容'` 分类（如果不再使用）。

### 修改后的对比表结构

| 分类 | 说明 |
|:-----|:-----|
| 基础信息 | 价格 + 体验包分发权数量（有差异） |
| 佣金权益 | 一级/二级佣金比例（有差异） |
| 可分成产品 | 11款产品（相同，展示完整性） |
| 专属权益 | VIP权益（有差异） |

这样用户的注意力会集中在真正有差异的地方：佣金比例和分发权数量。
