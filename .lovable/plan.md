

## 为"觉醒之旅"卡片添加训练营目标

### 目标

在卡片中增加一个清晰的"7天训练营目标"提示，让用户知道完成训练营后能达到什么水平。

### 设计思路

根据等级配置，完成7天训练营可以解锁 **Lv.4 ⭐ 信念转化者**（700积分）。在觉醒分数维度，训练营目标是达到 **高度觉醒（80+）**。

在核心数据区下方，增加一行简洁的目标提示：

```text
┌──────────────────────────────────────┐
│ 🎮 我的财富觉醒之旅             (i) │
│                                      │
│   🟡 稳步觉醒                       │
│   65 分  (成长 +0)                   │
│   距 🟢 高度觉醒 还差 15 分         │
│                                      │
│   🎯 7天目标：⭐ Lv.4 信念转化者    │  <-- 新增
│   ━━━━━━━━━━░░░░ 进度 65%           │  <-- 新增
│                                      │
│ 🌱──🌿──🌻──⭐──🌟──👑            │
│        ↑ 当前                        │
│   距 Lv.2 觉察学徒 还需 90 积分     │
└──────────────────────────────────────┘
```

### 技术细节

**修改文件：`src/components/wealth-camp/GameProgressCard.tsx`**

1. 在"距下一觉醒状态提示"和"等级图标轨道"之间，新增一个"7天目标区块"
2. 目标为 Lv.4（minPoints: 700），显示当前积分占 700 的进度百分比
3. 如果用户已达到或超过 Lv.4，显示"已达成"状态
4. 使用 `progress.total_points` 计算进度条：`Math.min(100, (totalPoints / 700) * 100)`

**新增代码逻辑**：
- 目标等级固定为 `awakeningLevels[3]`（Lv.4 信念转化者，700分）
- 进度 = `total_points / 700 * 100`
- 已达成条件：`total_points >= 700`
- 样式：amber 渐变进度条 + 目标 icon，与卡片整体风格统一
