

## 目标
统一所有 AI 教练语音通话页面（`CoachVoiceChat`）的圆形主按钮图标语义：
- **通话中**：显示 `Phone`（无斜线的电话图标，表示"通话进行中"）
- **未连接 / 已挂断 / 强制关闭中**：显示 `PhoneOff`（带斜线，表示"已断开"）

并对该按钮区做小幅可用性优化。

## 改动范围
仅一个共享组件，所有教练（小劲 ✨、财富觉醒教练 💎、劲老师、其他 AI 教练）自动同步生效：

- `src/components/coach/CoachVoiceChat.tsx`

## 详细改动

### 1. 翻转底部主按钮的图标判断（第 2933–2937 行）
```tsx
{status === 'connected' ? (
  <Phone className="w-6 h-6" />        // 通话中：无斜线
) : (
  <PhoneOff className="w-6 h-6" />     // 未连接/挂断：有斜线
)}
```

### 2. 同步修正顶部右上角"挂断"小按钮（第 2391 行）
该按钮始终是"挂断动作"，按用户语义统一使用 `PhoneOff`（保持现状即可，已带斜线，与文案"挂断"一致），无需调整。

### 3. 顺带优化建议（同一处按钮区）
- **强制关闭态视觉提示**：当 `isEnding === true` 时（用户已点过一次挂断、进入强制关闭分支），按钮内显示 `Loader2` 旋转 + `PhoneOff`，并将背景从 `bg-red-500` 切换为 `bg-red-600`，让用户明确感知"再次点击会强制关闭"。
- **无障碍**：给按钮补 `aria-label`：通话中为"挂断通话"，未连接为"开始通话"，强制关闭态为"强制关闭"。
- **防误触**：通话中按钮加 200ms 长按确认（仅在 `status === 'connected'` 且非 PTT 模式生效），避免微信小程序 WebView 误触秒挂。可作为可选项，默认不开启，等你确认再加。

## 不改动的内容
- PTT（按住说话）模式下的录音圆按钮（`VoiceInputButton`）逻辑不变，它语义是"按住说话"，与挂断图标无关。
- 顶部右上角挂断小按钮文案与图标保持一致（PhoneOff + "挂断"）。

## 验证方式
进入 `/xiaojin` → 点开"小劲"语音通话：
1. 未接通时：底部圆按钮 = 带斜线电话；
2. 接通后：底部圆按钮 = 普通电话（无斜线）；
3. 点一次挂断：进入强制关闭态，按钮显示 Loader + 带斜线电话；
4. 同步在 `/wealth-coach-voice`、其他教练入口验证一致。
