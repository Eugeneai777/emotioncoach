

# 优化「总时长」指标的清晰度

## 问题分析

当前「总时长 3 分钟」存在两个问题：

1. **语义不清**：用户无法判断是「本月总时长」还是「历史总时长」——实际是本月
2. **与其他指标重复**：汇总条已有「本月消耗 X 点」和「剩余 X 点」，再加一个「总时长」信息密度低且不直观（用户更关心点数而非分钟数）

## 方案

将「总时长 X 分钟」替换为更有价值的指标——**本月通话次数**，同时把原来右上角的「本月 X 次」移入汇总条统一展示，右上角改为显示「最近 20 条」等辅助信息。

或者，如果保留时长指标，改为明确标注「本月通话」并精确到秒：

| 现状 | 改后 |
|------|------|
| `总时长 3 分钟` | `本月通话 1分32秒` |

## 推荐方案：简化汇总条

汇总条保留 3 个指标，全部加「本月」前缀统一口径：

```text
🔋 剩余 70 点  ⚡ 本月消耗 24 点  📞 本月 3 次通话
```

去掉「总时长」——因为每条记录已经显示了各自的通话时长，汇总层面点数比时长更有业务意义。
