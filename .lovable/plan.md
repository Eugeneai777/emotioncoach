

# 身份绽放落地页优化：让用户一眼看懂"解决什么问题"

## 核心问题

团队反馈：当前痛点和优势描述太虚，用户不清楚训练营具体解决什么问题。需要把抽象概念转化为**具体可感的生活场景和解决方案**。

## 改动方案（仅改 `src/pages/IdentityBloomPromoPage.tsx`）

### 1. 重写痛点区域 — 从抽象到具体场景

**现在**（太虚）：
- "身份迷失 — 活得像别人的影子"
- "不敢拒绝 — 总在看别人脸色"
- "底气不足 — 总觉得自己不够好"

**改为**（具体症状 + 明确解法）：

| 痛点 | 描述 | 解法指向 |
|------|------|----------|
| 压力大睡不好，身体在报警 | 失眠、疲劳、头痛反复，吃药只治标 | → 身心整合，从根源改善 |
| 学了很多课，还是原地踏步 | 买了一堆书和课，听完激动、过后不动 | → 系统化方案，不再碎片学习 |
| 夫妻冷战、亲子疏远 | 想沟通却总吵架，越努力关系越僵 | → 关系工具，可操作的沟通方法 |
| 精力枯竭，效率越来越低 | 加班拼命却产出下降，身体越来越扛不住 | → 能量管理，而非时间管理 |
| 一个人扛着，没人真正理解 | 跟谁都说不出口，改变总是半途而废 | → 全程陪伴，教练+社群支持 |

每张卡片底部增加一行解法提示（如"→ 我们的解法：身心整合，从根源改善"），形成痛点到方案的即时闭环。

### 2. 新增"五大核心优势"区域 — 替代当前"科学深度设计"手风琴

删除 L633-L696 的手风琴折叠区域（用户大概率不会点开），替换为**外露的五大优势卡片**：

1. **系统化解决方案** — "不是碎片知识拼凑，而是从认知→情绪→身体→关系四维度层层递进的成长地图"
2. **专业身心整合** — "不只教心理学，更引入呼吸训练、神经调节等身体工作，打破焦虑-紧张的恶性循环"
3. **关系沟通工具** — "非暴力沟通、一致性表达等可操作工具，回家就能用"
4. **高效能量管理** — "基于生物节律的精力策略，用更充沛的状态创造更高价值"
5. **全程深度陪伴** — "专业教练引领 + 同伴社群 + 打卡复盘，确保真正落地"

每张卡片结构：标题 + 一句话解决什么 + 对应痛点标签。

### 3. 微调现有区域

- 痛点区域结语改为："这些不是你不够努力，而是缺少一套系统的解决方案。"
- "五大核心亮点"标题改为"你将拥有的全部资源"（与新增优势区域区分：优势=为什么有效，亮点=你拿到什么）
- 删除 `expandedHighlight` state（手风琴已移除）

### 4. 不改动的部分

四阶段课程、每日流程、教练团队、交付金字塔、价格、购买/兑换逻辑、分享功能等全部保持不变。
