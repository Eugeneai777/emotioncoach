

# 删除商品：知乐胶囊（健康食品分类）

## 操作
仅删除分类为"健康食品"的那条记录，保留"医疗保健"分类的记录。

| 操作 | ID | 分类 |
|------|-----|------|
| 删除 | `7a207697-6855-4868-95cc-4ff321193571` | 健康食品 |
| 保留 | `920ee870-35e1-4246-80ff-9421743060e5` | 医疗保健 |

## 技术细节
```sql
DELETE FROM health_store_products 
WHERE id = '7a207697-6855-4868-95cc-4ff321193571';
```

无需修改前端代码，删除后刷新页面即可生效。
