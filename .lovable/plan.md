

# 在 /zhile-products 页面新增协同套餐合集

## 现状

当前 `/zhile-products` 页面只有"按人群推荐"的选择器模式。两款协同套餐（情绪解压 + 财富觉醒）虽然散落在个别人群推荐中，但没有独立的集中展示位。

## 两款产品概要

| 套餐 | 路由 | 核心卖点 |
|------|------|---------|
| 心智×身体 协同抗压套餐 | `/promo/synergy` | 21天情绪解压训练营 + 知乐胶囊，针对职场压力/夫妻关系/亲子代沟 |
| 财富×身心 觉醒套餐 | `/promo/wealth-synergy` | 7天财富觉醒训练营 + 知乐胶囊，针对中年男性财务焦虑/决策力下降 |

## 合集命名

**「身心深度修复计划」**

副标题：训练营 + 知乐胶囊 · 双引擎协同 · 从根源解决问题

理由：两款产品共同特征是"心智训练 + 草本调理"的双通道深度方案，"深度修复"传达了非表面、系统性解决的价值感。

## 修改方案

### 文件：`src/pages/ZhileProductsPage.tsx`

在人群选择器（`你是哪类人群？`）**上方**，插入一个独立的合集区块：

```text
┌─────────────────────────────────┐
│  🔬 身心深度修复计划             │
│  训练营+知乐胶囊·双引擎协同      │
├─────────────────────────────────┤
│ ┌─────────────┐ ┌─────────────┐ │
│ │ 🧠 情绪解压  │ │ 💰 财富觉醒  │ │
│ │ 协同抗压套餐 │ │ 身心觉醒套餐 │ │
│ │ 训练营+胶囊  │ │ 训练营+胶囊  │ │
│ │   限时特惠   │ │   限时特惠   │ │
│ │    查看 →    │ │    查看 →    │ │
│ └─────────────┘ └─────────────┘ │
└─────────────────────────────────┘
```

- 两张卡片并排（`grid grid-cols-2`），点击跳转对应推广详情页
- 卡片样式与页面暗色主题一致，带渐变边框高亮
- 每张卡片展示：emoji、套餐名、一句话卖点、"限时特惠"标签、箭头引导

### 不改动

- 下方的人群选择器和已有推荐列表保持不变
- 不影响胶囊购买流程
