
# 优化 WealthAdvisorQRCard - 增强扫码行动呼召

## 目标
重新设计卡片，让"扫码添加顾问"的行动号召更强烈、更有吸引力，突出两大核心价值（7天觉醒路径 + 随时觉醒对话），营造紧迫感。

## 修改文件
`src/components/wealth-block/WealthAdvisorQRCard.tsx`

## 设计优化方案

### 1. 强化标题区 - 制造紧迫感
- 标题改为更有行动感的文案："立即获得你的专属觉醒方案"
- 副标题加入限时/稀缺性暗示："仅限测评用户 · 免费领取"
- 加大标题字号，使用渐变文字效果

### 2. 痛点区 - 加强刺痛感
- 保留个性化卡点+模式展示
- 痛点文案后追加一句行动导向语："现在就有机会改变"

### 3. 两大价值卡片 - 视觉升级
- 图标尺寸加大（w-11 h-11），颜色更鲜明
- 文案更具体化、更有获得感：
  - "7天觉醒路径" 改为 "7天定制觉醒路径"，副文案加"马上获得"
  - "随时觉醒对话" 改为 "1对1随时觉醒对话"，副文案强调"不限次数"
- 卡片加入微动画（hover/pulse 边框效果）

### 4. 二维码区 - 最大化行动呼召
- 二维码尺寸增大（w-44 h-44）
- CTA 文案改为更直接的："长按识别，马上领取"
- CTA 文字加大加粗，使用脉冲动画吸引注意
- 底部社会证明数字更新："已有 3,200+ 人领取"
- 在二维码上方添加指向箭头动画

### 5. 整体视觉
- 外层发光边框动画加强（更亮的渐变色）
- 背景增加细微的星光粒子效果（用 CSS 伪元素）

## 技术细节
- 仅修改 `WealthAdvisorQRCard.tsx` 一个文件
- 使用 Tailwind CSS 动画类 + framer-motion
- 不改变组件接口（props 不变）
- 不新增依赖
