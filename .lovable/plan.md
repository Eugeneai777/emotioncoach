

## 优化训练营卡片 (TrainingCampCard)

### 当前问题

从截图看，卡片存在以下可优化点：

1. **里程碑图标偏大且灰度效果不佳** — 未达成的图标（⭐、🏆）显示为灰色轮廓，视觉不够精致
2. **进度条与里程碑脱节** — 图标在上面均匀分布，进度条在下面，但 position 没有视觉对齐
3. **按钮区域比例失调** — "查看记录"按钮占大部分宽度，"介绍"按钮偏小且边框样式突兀
4. **缺少目标感** — 只显示"已完成 1/7 天"，没有告诉用户完成后能获得什么

### 优化方案

```text
┌──────────────────────────────────────┐
│ 🏕️ 财富觉醒训练营       🔥 连续1天 │
│ 第2天 · ✅ 今日已打卡               │
│                                      │
│ 🌱 启程 ────── ⭐ 中期 ────── 🏆 毕业│
│ ━━━━━━━░░░░░░░░░░░░░░░░░░░░░░ 14%  │
│                                      │
│ 🎯 毕业目标：觉醒分达到 XX 分       │  <-- 新增
│                                      │
│   [📅 查看记录]      [📈 介绍]      │
└──────────────────────────────────────┘
```

### 技术细节

**修改文件：`src/components/camp/TrainingCampCard.tsx`**

1. **里程碑与进度条整合**：将图标放在进度条上方对应位置（使用 absolute positioning），让视觉关联更紧密
2. **缩小图标**：从 `text-lg` 改为 `text-base`，未达成状态从灰度改为 `opacity-30`
3. **按钮均等宽度**：两个按钮都设为 `flex-1`，视觉平衡
4. **新增毕业目标行**（仅财富训练营）：
   - 查询 `user_awakening_progress` 获取 `baseline_awakening`
   - 计算目标：`Math.min(baseline + 20, 95)`
   - 显示一行文字："🎯 毕业目标：觉醒分达到 XX 分"
   - 如果没有基线数据则不显示（兼容非财富训练营）
