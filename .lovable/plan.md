

## /promo/midlife-men-399 落地页商业逻辑重构

### 一、问题诊断

**现状文案**："¥399 学费，可全额抵扣 ¥3980 闭门修复计划"

**三个商业漏洞**：
1. **数学不通**：¥399 全额抵扣 ¥3980，等于白送，用户立刻警觉"是不是套路"
2. **价值锚点缺失**：没有出现 ¥4180 原价，¥3980 显得是凭空定价，没有"省了多少"的获得感
3. **筛选机制被弱化**：真实业务里 ¥800 抵扣券是"完赛门槛 + 付费意愿验证"，这是高端营的核心筛选器，现在完全没体现

### 二、商业模型重新表达（核心逻辑）

把真实路径包装成用户**听得懂、信得过、有获得感**的三段式：

```
¥4180  闭门修复计划原价（锚点）
 ↓ 完赛 ¥399 体验营 解锁"老学员价"
¥800   购买 ¥1000 抵扣券（资格 + 诚意金）
 ↓
¥3980  老学员专属抢订价（省 ¥200 + 锁席位）
```

**关键叙事转向**：
- 不再说"¥399 抵扣 ¥3980"（数学不通）
- 改说"完赛 ¥399，解锁 ¥800 换 ¥1000 的老学员资格券"（机制清晰）
- 强调 ¥4180 → ¥3980 是**老学员专享省 ¥200 + 200 个限定席位**（稀缺 + 获得感）

### 三、08 模块新结构（升舱钩子）

**Eyebrow**：如果 7 天后你想「正经处理」这件事
**Title**：完赛即解锁「老学员专属通道」

**主卡内容（三层递进）**：

1. **顶部叙事**（一句话定位）
   > ¥399 体验营，是「中年男性闭门修复计划」的入场资格筛选。
   > 完赛学员，进入 200 人老学员通道。

2. **价格阶梯表**（视觉化锚点，最关键）
   ```
   ┌─────────────────────────────────────┐
   │ 闭门修复计划 · 公开报价     ¥4,180  │
   │ ─────────────────────────────────── │
   │ 老学员资格券（¥800 购 ¥1000）       │
   │ ─────────────────────────────────── │
   │ 你的实际抢订价              ¥3,980  │
   │                       (省 ¥200 + 优先锁席) │
   └─────────────────────────────────────┘
   ```

3. **三条权益**（替换原"全额抵扣"）
   - ✓ 完赛即获 老学员资格券购买权（¥800 → ¥1000 抵扣）
   - ✓ 优先锁定本期 200 个限定席位（不公开报名）
   - ✓ 由总教练 1 对 1 路径定制（先评估，后入营）

4. **底部金句**
   - —— ¥399，是诚意；¥800，是承诺
   - —— 闭门修复计划，只给已经走完 7 天的人

5. **CTA**：先看看 ¥3980 闭门修复计划 ↗（不变）

### 四、09 模块价格区微调

**当前**："含全部交付 · 学费可全额抵扣 ¥3980 闭门修复"

**改为**："含全部 7 天交付 · 完赛解锁 ¥3,980 老学员抢订资格"

去掉"全额抵扣"这种容易被算账拆穿的措辞，换成"解锁资格"这种**机制型表达**。

### 五、为什么这样改更高转化

| 维度 | 旧版 | 新版 |
|---|---|---|
| 可信度 | ¥399 抵 ¥3980（疑似套路） | ¥4180 → ¥3980（清晰锚点） |
| 获得感 | "全额抵扣"反而像白送 | "省 ¥200 + 锁席位"具体可感 |
| 筛选感 | 无 | "完赛资格 + ¥800 诚意金" 双门槛 |
| 高端感 | 弱（送的越多越掉价） | 强（资格制 + 200 限定席） |
| 数学一致 | 用户算账会发现矛盾 | 用户算账后更想买（¥800 换 ¥1000） |
