## 内容结构（9大模块）

### 模块1：为什么记录能改命
- 很多人以为记录是玄学
- 神经科学研究：每天遗忘70%+
- 大脑默认沿用前一天模式

### 模块2：自动驾驶状态
- 差不多的起床方式
- 差不多的工作节奏
- 差不多的刷手机、娱乐
- 差不多的情绪反应
- 结果：忙却不清楚在忙什么

### 模块3：记录的科学价值
- 持续记录者5年后：决策速度更快
- 职业转型效率提升2-3倍
- 不是更聪明，而是更清醒

### 模块4：极简记录方式（核心）
**① 困境记录**
- 不是写"今天遇到糟糕的事"
- 而是写"破局关键点"或"命运转折点"
- 这样写你会看见：惯性情绪反应、错误处理方式
- 觉察发生 → 跳出旧模式

**② 顺境记录**
- 散步、电影、一顿饭、聊天、灵感、平静瞬间
- 累积"我在好好生活"的证据
- 回头翻看不会觉得一无所获

### 模块5：觉察入口做的三件事
1. 把无意识变成可觉察
2. 把情绪反应变成可调整
3. 把碎片时间变成成长复利

### 模块6：为什么能坚持
- 很快且持续的正向反馈
- 第一天记录焦虑就明显下降
- 不是问题消失，而是不再被困在混乱里

### 模块7：为什么是6大入口（完整版）
**7.1 六类问题映射表**

| 你以为的问题 | 本质属于 |
|-------------|---------|
| 情绪失控、焦虑 | 情绪系统 |
| 空虚、匮乏、无力 | 价值与滋养系统 |
| 拖延、忙但没进展 | 行动系统 |
| 犹豫、反复纠结 | 决策系统 |
| 吵架、冷战、误解 | 关系系统 |
| 迷茫、没方向 | 意义与方向系统 |

**7.2 为什么不做更多入口？**
- 超过7个选项 → 决策疲劳 → 什么都不做
- 6个是认知负担最小且不遗漏核心的数量

**7.3 觉察闭环**
情绪 → 感恩 → 行动 → 选择 → 关系 → 方向 → 情绪...
- 情绪：告诉你哪里出问题
- 感恩：修复神经系统
- 行动：把意识落到现实
- 选择：关键点做清醒判断
- 关系：修复与世界的连接
- 方向：对齐长期人生

**7.4 为什么每天都值得记录**
- 情绪：每天都有，很少停下来识别
- 感恩：每天都在消耗，很少滋养
- 行动：每天都在忙，不清楚为什么
- 选择：每天都在决定，以为自己是理性的
- 关系：每天都在互动，很少真正沟通
- 方向：每天都在向前，不知道往哪走

**7.5 干预层级表**

| 入口 | 干预层级 |
|-----|---------|
| 情绪 | 神经反应与情绪回路 |
| 感恩 | 注意力分配与神经可塑性 |
| 行动 | 行为习惯与执行系统 |
| 选择 | 潜意识决策机制 |
| 关系 | 社会连接与安全感 |
| 方向 | 意义感与长期动机 |

**7.6 为什么不是其他名字/维度**
- 财富、学习、健康是结果变量
- 财富 = 行动 x 选择 x 关系 x 方向
- 如果把结果当入口，会焦虑、内疚、放弃

### 模块8：6入口分类展示
**困境 → 破局关键点**
- 🔥 情绪 - 看见被忽略的信号
- 🧩 选择 - 理清内心的冲突
- 🤝 关系 - 表达未被听见的话

**顺境 → 滋养与锚定**
- 💛 感恩 - 重写神经回路
- ⚡ 行动 - 把碎片变成复利
- 🌟 方向 - 点亮人生主线

### 模块9：开始觉察的召唤
- 你不需要想清楚
- 也不需要写得好
- 只要写下一点真实的状态
- 觉察入口会陪你把混乱变成清晰

**一句话总结（给用户）**
> 人生看似复杂，其实每天只在这6个地方卡住。
> 觉察入口，就是把这6个地方一一照亮。

**灵魂金句**
> 6大入口不是在管理你的人生，
> 而是在你每天最真实的时刻，
> 给你一个可以回到清醒的位置。
