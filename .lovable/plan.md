

## 将"绑定用户"更名为"负责人"

在 4 个文件中替换文案：

### 1. `BindUserDialog.tsx`
- 标题：`绑定用户账号` → `设置负责人`
- 说明：`输入用户手机号，将其账号与此行业合伙人关联。绑定后合伙人可登录访问合伙人中心。` → `输入负责人手机号，设置后该负责人可登录访问合伙人中心。`
- 按钮：`确认绑定` → `确认设置`

### 2. `IndustryPartnerList.tsx`
- 表头：`绑定用户` → `负责人`
- 状态：`已绑定` → 保留显示昵称
- 按钮：`绑定` → `设置`
- 确认框：`确认解除绑定？` → `确认移除负责人？`

### 3. `PartnerInfoEditor.tsx`
- 卡片标题：`账号绑定` → `负责人`
- 状态：`已绑定用户账号` → `已设置负责人`
- 按钮：`解除绑定` → `移除负责人`
- 确认框：`确认解除绑定？解除后该合伙人将无法通过合伙人中心访问。` → `确认移除负责人？移除后该合伙人将无法通过合伙人中心访问。`
- 提示：`已解除绑定` → `已移除负责人`
- 空状态：`尚未绑定用户账号，绑定后合伙人可登录访问合伙人中心。` → `尚未设置负责人，设置后负责人可登录访问合伙人中心。`
- 按钮：`绑定用户` → `设置负责人`

### 4. `useIndustryPartners.ts`
- Toast：`已绑定用户: xxx` → `已设置负责人: xxx`
- Toast：`绑定失败` → `设置失败`
- Toast：`已解除绑定` → `已移除负责人`
- Toast：`解绑失败` → `移除失败`
