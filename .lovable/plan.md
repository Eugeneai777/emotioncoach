

## 在觉醒之旅卡片中显示个性化目标数字

### 问题

测评结果页展示了清晰的目标数字（如"7天目标 75+"、"毕业目标 85+"），但进入训练营后的"觉醒之旅"卡片中，这些个性化目标消失了。用户失去了明确的努力方向。

### 方案

利用已存储的 `baseline_awakening`（觉醒起点），在 `GameProgressCard` 中用 **同样的公式** 计算并显示 7 天目标数字，让用户在训练营中始终看到自己的目标。

目标公式（与测评结果页一致）：
- 7 天目标 = `min(baseline_awakening + 20, 95)`

### 技术细节

**修改文件：`src/components/wealth-camp/GameProgressCard.tsx`**

1. **替换现有的 7 天目标区块**（当前显示的是 Lv.4 / 700 积分目标）：
   - 改为基于 `baseline_awakening` 计算的觉醒分目标：`day7Target = Math.min(baseline_awakening + 20, 95)`
   - 显示三个关键数字：起点分 → 当前分 → 目标分
   - 进度条从起点到目标的百分比：`(current - baseline) / (target - baseline) * 100`

2. **目标达成状态**：
   - 未达成：显示"距目标还差 X 分"
   - 已达成：显示"已达成 7 天目标"

3. **移除"距下一觉醒状态"提示**（与目标区信息重复）

4. **数据来源**：直接使用 `progress.baseline_awakening` 和 `progress.current_awakening`，无需额外数据库查询
