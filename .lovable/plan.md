

## 顶级对话设计重构 PTT 通话界面

### 设计参考
ChatGPT Voice / Gemini Live / Siri 的核心法则：**一屏一焦点**。屏幕上只允许存在三个层次：
1. 动态视觉（说话状态可视化，居中、唯一）
2. 实时字幕（最重要的信息）
3. 单一操作（PTT 按钮）

所有装饰、铭牌、提示、徽章都必须从主视觉中消失，降级为顶部细线信息。

### 当前问题 vs 顶级设计对比

| 元素 | 当前 | 改造后 |
|---|---|---|
| 心形大头像 + 双层呼吸光环 | 占屏 1/3 | **删除**，由"动态光球"替代 |
| 「有劲AI生活教练」大标题 | text-xl 占整行 | **删除**（顶部已有挂断/计时足够） |
| 「8点/分钟·余额…」长副标题 | 一长行 | **删除**（顶栏已有 16点 计费显示） |
| 音波条（4 根静态柱） | 居中显眼 | **删除**（光球本身即是状态） |
| 网络延迟橙色横条 | 整行 banner | **删除主区**，仅保留顶栏小图标 |
| 「正在聆听...」+ 麦图标 | 单独一行 | **删除**（光球颜色变化即表达） |
| 字幕区 | 「你：」「教练：」前缀 | 去掉前缀，靠颜色/位置区分 |
| PTT 大圆按钮 | 底部边缘 | **上移到屏幕 70% 处**，进入拇指区 |

### 改造方案 — `src/components/coach/CoachVoiceChat.tsx`

**1. 顶栏（L2064–L2143）保持极简单行，移除冗余**
- 中间合并为一行：`1:04 · 余额 16点 · 📶`（网络小图标，点击 toast 详情）
- 删除 `ConnectionStatusBadge` 大徽章渲染（242ms 数字显示在 toast 即可）
- 字号统一 `text-xs text-white/50`

**2. 主区（L2145–L2247）重写为「光球 + 字幕 + 留白」三层**

```text
┌──────────────────────────┐
│ ←  1:04 · 16点 · 📶  挂断 │  顶栏
│                          │
│                          │  
│        ╭─────╮           │
│       │ 光球 │            │  ← 唯一动态视觉
│        ╰─────╯           │
│                          │
│                          │
│   嗯，感觉有点迷茫…         │  ← 用户字幕（淡白）
│   也许可以从小处着手…       │  ← AI 字幕（玫红，光标）
│                          │
│                          │
│         ╭───╮            │
│        │ 🎤 │             │  ← PTT 主操作（上移到 70%）
│         ╰───╯            │
│        按住说话           │
└──────────────────────────┘
```

**3. 「光球」组件**（替代心形头像 + 音波条 + 状态文字）
- 一个 80px 的圆形 gradient blob（rose-400 → rose-600），带高斯模糊光晕
- 状态映射（颜色 + 动画即语义，无需文字）：
  - `idle/connected`：缓慢呼吸（scale 1 → 1.05，2.4s 周期），rose 50% 饱和
  - `user-speaking`：scale 跟随实时音量（用现有 audio level），饱和度提升，绿色光晕外环
  - `assistant-speaking`：scale 律动（流式波动），rose 高饱和 + 玫红光晕扩散动画
- 这一个元素同时承载了「头像」「音波条」「状态文字」三件事

**4. 字幕区（位于光球下方）**
- 移除「你：」「教练：」前缀
- 用户行：`text-white/55 text-sm`
- AI 行：`text-rose-100 text-[17px] font-medium leading-relaxed`（视觉重心）
- 仅保留最近一轮对话；空闲态显示淡灰提示「按住下方按钮 · 和教练说说」
- max-w-md 居中，px-6，min-height 占位防跳动

**5. 网络提示降级**
- 删除主区 `InCallNetworkHint` 橙色横条
- 网络异常时，顶栏 `📶` 图标变橙/红；点击 → toast `网络延迟 242ms · 建议靠近路由器`

**6. PTT 按钮区位置上移（L2520+）**
- 容器 `p-6 pb-safe` → `pb-[12vh]`，按钮位于屏幕约 70% 处，进入自然拇指区
- 按钮下方仅一行小字 `按住说话` / `松开发送`，移除其他副文案

**7. 删除/隐藏**
- `coachTitle / coachEmoji / coachSubtitle` 在 PTT 模式下不渲染
- 余额/费率长副标题（顶栏已有简化版）
- 音波条 `AudioWaveform`（PTT 模式下不渲染）
- 「正在聆听 / 你正在说话 / 劲老师正在回复」状态文字（光球已表达）
- 主区 `InCallNetworkHint`

### 涉及文件
