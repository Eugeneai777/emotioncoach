

# 新建"协同抗压"推广页 — 不同设计风格

## 思路
创建一个全新页面 `SynergyPromoPage.tsx`，独立于现有 `PromoPage`，使用**深色科技感 + 数据驱动**的设计风格（vs 现有的暖橙色），专注传达"心智 + 身体 = 全天候抗压"的核心理念。

## 新页面设计风格对比
- 现有页面：暖橙色渐变，情感导向
- 新页面：**深蓝/紫色科技风**，数据导向，强调科学实证

## 页面结构

### 1. Hero — 公式视觉冲击
大字体公式：`🧠 练心智 × 💊 护身体 = 24h 全天候抗压`
背景使用深蓝紫渐变，配合粒子动画。

### 2. 核心问题场景（痛点共鸣）
3个职场压力场景卡片，每个场景配图标和数据：
- 开会焦虑（78%职场人有会议恐惧）
- 失眠循环（65%因工作压力失眠）
- 午后崩溃（83%下午效率暴跌）

### 3. 双引擎解决方案
左右对比布局，训练营和知乐各自角色 + 大产品图：
- 训练营：心理层面，认知重塑，习惯养成
- 知乐胶囊：生理层面，即时缓解，身体修复
中间用"+"连接，底部强调"缺一不可"。

### 4. 协同效果数据看板（核心区块）
深色卡片，4组对比数据带进度条动画：
- 压力缓解：60% vs 55% vs **92%**
- 起效速度、睡眠质量、情绪稳定度、工作效率
- 每组用渐变进度条，组合数据高亮金色

### 5. 24小时防护时间轴（横向滚动设计）
从早到晚的时间轴，圆形节点，区分心智/身体两条线：
```text
7:30 训练营 → 8:00 胶囊 → 12:30 胶囊 → 19:00 胶囊 → 全天覆盖
```

### 6. 产品实拍 + 规格数据
知乐胶囊大图（占满宽度），配合产品规格：
- 每瓶90粒 / 每日3粒 / 30天用量
- 主要成分和功效说明

### 7. 用户见证（数据化）
每条见证配具体改善数据：
- "焦虑评分从8.2降到3.1"
- "睡眠时长从5h增加到7.5h"

### 8. 底部CTA
固定底部，深蓝渐变按钮

## 实现
1. 创建 `src/pages/SynergyPromoPage.tsx` — 全新独立页面，硬编码内容
2. 在 `src/App.tsx` 添加路由 `/promo/synergy`
3. 复用现有 `UnifiedPayDialog` 和产品图资源
4. 不涉及数据库改动
