# 青春期家长必修课 · 海报商业策划方案

## 一、对你思路的评估（先诊断，再优化）

### ✅ 做得好的地方

1. **人设有"生命厚度"**——27年与疾病共舞 + 自学心理学历，是稀缺的"伤痕教练"叙事，强信任锚点。
2. **承诺具体**——"从对抗到对话"是家长能感知的结果。
3. **赠品有钩子**——读书会附加值合理。

### ⚠️ 需要优化的5个问题


| #   | 现状问题                                            | 优化建议                                                            |
| --- | ----------------------------------------------- | --------------------------------------------------------------- |
| 1   | **主标题太长太平**（"青春期家长必修课：10次家庭深度咨询…"），信息密度高但缺少痛点钩子 | 改成痛点开头：**"孩子一回家就关门？10次深度陪伴，把'对抗'谈成'对话'"**                       |
| 2   | **"原价8000限时6000"**对家长是模糊数字，缺锚定与紧迫感              | 拆成单次锚定：**"原价8000元（800元/次）→ 现优惠6000元（600元/次）· 仅限前10位"**          |
| 3   | **人设原文是第二人称叙述**，海报上不能直接放"你是…"，需改成第一人称/第三人称卡片化   | 重构成3张"信任卡片"：① 27年与疾病共舞 ② 自学拿下心理咨询学历+职业资格证 ③ "我知道这种感觉"——蹲下来的倾听者  |
| 4   | **缺"交付内容清单"** —— 用户只看到"10次"，不知道每次做什么            | 补充 4 行交付明细（见下方文案）                                               |
| 5   | **两款海报差异化不清晰**                                  | 明确定位：海报A=**成交转化海报**（带支付码，朋友圈/私聊用）；海报B=**引流加微海报**（带个微码，公域/社群破冰用） |


### 💡 额外建议

- 海报A正下方写"扫码支付即开通"，海报B写"先加微信1v1聊聊孩子情况"——动作指令必须明确。
- 生活照建议做**圆角/椭圆遮罩 + 柔光描边**，避免书架背景与海报主色冲突。

## 三、设计方案（两款）

### 通用规格

- 尺寸：**1080 × 1620 px**（3:4 朋友圈/小红书标准海报）
- 风格：**温柔治愈系编辑风**，避免廉价微商感
- 配色：莫兰迪绿米 + 暖原木 + 墨绿点缀（呼应生活照里的绿色长裙与木色书架，视觉自然延展）
  - 主背景 `#F5F1E8`（米杏）
  - 强调色 `#5C7A5A`（墨绿）
  - 文字主色 `#2A2E2A`
  - 价格高亮 `#C84B31`（赤陶红，仅用于划线价/限时数字）
- 字体：标题 **思源宋体 Bold**（厚重信任感）+ 正文 **思源黑体 Regular**

## 四、执行方式

使用 Python + Pillow（PIL）合成两张海报 PNG，输出到 `/mnt/documents/`：

- `poster-A-pay.png`（成交版，嵌入支付二维码）
- `poster-B-wechat.png`（引流版，嵌入个人微信二维码）

生活照将做**圆角矩形 + 内描边 + 轻投影**处理，与米杏底色自然融合；二维码下方加白色圆角卡片承托，确保扫码识别率。

完成后两张图片直接以 `<presentation-artifact>` 形式交付，可下载、可迭代 `_v2`。
