

# 触动人心的文案与内容优化

## 核心理念

当前页面文案偏"功能描述"（做什么），缺少"情感共鸣"（我懂你）。优化方向：**先共情，再引导**。让妈妈感觉"这个页面懂我"，而不是"这个页面在卖东西给我"。

## 具体改动

### 1. MamaHero — 从"问问题"变成"温暖拥抱"

**现在**：`宝妈AI生活助手` + `今天最困扰的是什么？`
**问题**：像问卷调查，冷冰冰

**优化为**：
- 标题：`嘿，妈妈` （亲切、直接）
- 副标题：`今天辛苦了，我在这里陪你 💛`（先共情）
- 场景按钮文案更感性：
  - `😤 孩子不听话` → `😤 又跟孩子吵了`
  - `📚 不爱学习` → `📚 催作业催崩了`  
  - `😩 今天很累` → `😩 好累，想躺平`
  - `💬 沟通不好` → `💬 说什么都没用`
  - `🌫️ 有点迷茫` → `🌫️ 不知道为了什么`

### 2. MamaEmotionCheck — 更像一个"安全的出口"

**现在**：`情绪急救站` + `30秒释放情绪，给自己一个拥抱`
**问题**：还可以，但"急救站"略医疗化

**优化为**：
- 标题：`此刻，你还好吗？`
- 副标题：`不需要坚强，这里可以放下所有`
- 视觉：加一个温暖的拥抱感文字

### 3. MamaDailyEnergy — 从"鸡汤"升级为"被看见"

**现在**：`今日妈妈一句话` + 感恩记录
**问题**：标题平淡，感恩placeholder太具体

**优化为**：
- 标题：`💌 今天，想对你说`（像一封信）
- 感恩区标题：`记录一个让你微笑的瞬间`
- Placeholder：`也许是孩子突然说了句暖心话...`
- 提交按钮：`记下这份温暖 ✨`

### 4. MamaAssessmentEntry — 能量检测更有代入感

**现在**：`今日能量检测` + `每天1分钟，为5个维度打分`
**问题**：像体检报告，缺乏吸引力

**优化为**：
- 标题：`今天的你，能量还够吗？`
- 副标题：`1分钟了解自己的状态，找到最需要的支持`
- 按钮：`看看我的能量 →`
- 底部提示改为：`测完后会推荐最适合你的工具和支持`

### 5. MamaCampEntry — 用故事感召

**现在**：`21天亲子关系训练营` + `每天15分钟，系统提升亲子沟通能力`
**问题**：像广告文案

**优化为**：
- 标题：`21天，和孩子重新靠近`
- 副标题：`不是学技巧，而是找回你和孩子之间的温度`
- 标签保留，但按钮改为：`我想试试 →`

### 6. 悬浮按钮 — 更温暖

**现在**：`找教练聊聊`
**优化为**：`想找人说说话 💬`

## 改动文件清单

| 文件 | 改动 |
|------|------|
| `MamaHero.tsx` | 标题、副标题、场景按钮文案 |
| `MamaEmotionCheck.tsx` | 标题、副标题文案 |
| `MamaDailyEnergy.tsx` | 标题、感恩区文案、placeholder、按钮 |
| `MamaAssessmentEntry.tsx` | 能量检测标题、副标题、按钮、底部提示 |
| `MamaCampEntry.tsx` | 标题、副标题、按钮文案 |
| `MamaAssistant.tsx` | 悬浮按钮文案 |

所有改动仅涉及文案内容，不改变功能逻辑和路由。
