

## 仅首屏展示底部信息

### 需求理解

用户希望底部信息（公众号链接、付费提示、ICP备案）仅在答题的**第一题**时显示，后续题目不显示。

---

### 解决方案

在 `WealthBlockQuestions` 组件中，根据 `currentIndex === 0` 条件渲染底部信息区域。

---

### 修改文件

#### 文件: `src/components/wealth-block/WealthBlockQuestions.tsx`

**修改位置：在导航按钮区域下方，添加条件渲染的底部信息**

```tsx
{/* 导航按钮 - 移动端 sticky 底部 */}
<div className="flex gap-3 pt-4 sm:pt-6 mt-auto sticky bottom-0 ...">
  {/* 上一题/下一题按钮 */}
</div>

{/* 仅首屏（第一题）显示底部信息 */}
{currentIndex === 0 && (
  <div className="mt-6 pt-4 border-t space-y-2 text-center pb-[env(safe-area-inset-bottom)]">
    {/* 关注公众号链接 */}
    <a 
      href="https://mp.weixin.qq.com/s/your-wechat-url"
      target="_blank"
      rel="noopener noreferrer"
      className="text-blue-500 text-sm block"
    >
      点此关注公众号
    </a>
    
    {/* 付费提示（红色） */}
    <p className="text-red-500 text-xs">
      需付费后方可查看结果，结果纯属娱乐仅供参考
    </p>
    
    {/* 公司信息和ICP备案 */}
    <p className="text-muted-foreground text-xs">
      北京好企劲商务信息咨询有限公司 京ICP备2023001408号-5
    </p>
  </div>
)}
```

### 技术要点

| 要点 | 说明 |
|------|------|
| 条件渲染 | `currentIndex === 0` 仅在第一题时显示 |
| 非固定定位 | 使用普通 flow 布局，随页面滚动（不是 fixed） |
| 安全区域 | `pb-[env(safe-area-inset-bottom)]` 适配 iPhone 底部 |
| 分隔线 | `border-t` 与导航按钮区分开 |
