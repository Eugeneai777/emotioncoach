

# ¥389 知乐合作引流款售前页 — 商业评估与设计方案

## 一、商业评估

### 你的想法很清晰，有几个要点值得确认

**做得对的：**
- ¥389 作为引流款定价合理，在"冲动购买"阈值内（<500），同时包含实物（知乐胶囊）+服务（1V1教练），感知价值远超价格
- 核心亮点选"1V1教练 + 知乐胶囊"是正确的，因为这两样都是高感知稀缺品，不像AI教练/社群可以免费体验到
- 用1V1教练辅导做后续3980身份绽放训练营的引流入口，符合"先体验后承诺"的转化逻辑

**需要优化的：**
1. **海沃塔沙龙确实应该去掉** — 它和1V1教练定位冲突（一个是"专属深度"，一个是"团队讨论"），且你说转化率不高。389引流款的核心逻辑是"让用户感受到1V1的价值"，海沃塔会稀释这个焦点
2. **"30分钟1V1深度咨询"应该作为全页最大卖点**，不是和其他服务并列。建议用"30分钟 = 解决3个月情绪内耗"这种高对比度的价值锚定
3. **赠品（学员服务群）应该弱化呈现**，避免和核心卖点抢注意力。放在"你还将获得"区域即可

### MVP阶段商品定位建议

```text
¥389 引流款定位公式：
"一次30分钟1V1深度咨询 + 一瓶知乐胶囊"
= 让用户用最低成本，体验到"被专业教练看见"的感觉
= 为3980身份绽放训练营建立信任和需求
```

## 二、页面设计方案

### 新建文件：`src/pages/ZhileCoachPromoPage.tsx`
路由：`/promo/zhile-coach`

基于现有 `SynergyPromoPage.tsx` 的成熟架构（Hero + 痛点 + 亮点 + 教练 + 产品 + 价格 + 兑换），针对389引流款重新设计内容。

### 页面结构（从上到下）

**1. Hero 区域 — 直击核心卖点**
- 主标题："30分钟，解开你3个月的情绪内耗"
- 副标题："知乐 × 有劲 联合推出 · 1V1教练深度咨询 + 知乐草本胶囊"
- 价格锚定：~~原价¥899~~ → ¥389（限量体验价）
- CTA按钮：兑换码激活

**2. "你是不是这样" — 3个精准痛点场景**
（比399页面更聚焦，只保留3个最能引发共鸣的）
- 😶 情绪内耗：焦虑、易怒、失眠，吃药不管用，不知道问题出在哪
- 🤐 一个人扛着：想找人聊聊，又怕被评判，朋友劝你"想开点"更烦
- 💊 试过很多方法：买课、看书、冥想APP，但坚持不下来，问题还在

**3. 核心卖点区 — "你将获得"**（全页最重要的模块）

卡片1（大卡，占全宽）：
- 🎯 **30分钟1V1教练深度咨询**
- "不是聊天，是专业诊断"
- 三个具体交付：① 个人情绪卡点精准诊断 ② 定制化解决方案 ③ 后续成长路径规划
- 底部标注："由资深生命教练亲自执行，非AI替代"

卡片2（中卡）：
- 💊 **知乐胶囊 1瓶**
- "从身体根源调理情绪与睡眠"
- 香港HKC认证 · 16味草本 · 不含激素

**4. "30分钟能解决什么" — 价值可视化**
用对比格式展示：
| 你现在的状态 | 30分钟后你会得到 |
|---|---|
| 不知道自己为什么焦虑 | 清晰的情绪卡点诊断报告 |
| 试了很多方法没用 | 针对你个人的定制方案 |
| 不知道下一步该怎么做 | 明确的成长路径规划 |

**5. 教练团队（精简版）**
只展示2-3位核心教练（黛汐 + 晓一 + Amy），不需要全员展示。强调"平均10年+经验"。

**6. 知乐胶囊模块（复用现有）**
复用 SynergyPromoPage 的胶囊展示区（产品图 + 规格 + 资质认证），微调文案。

**7. 赠品区 — "你还将获得"**（轻量展示）
- 📱 学员专属服务群（1V1服用指导 + 情绪管理干货 + 冥想内容）
- 标注"赠品"标签，视觉权重低于核心卖点

**8. 价格区 + CTA**
- ~~¥899~~ → ¥389
- "含1次30分钟1V1教练咨询 + 1瓶知乐胶囊 + 学员服务群"
- 兑换码激活按钮
- 底部加一句引导："体验后可升级完整身份绽放训练营（¥3980）"

**9. 吸底栏**
- 左侧：¥389 限量体验价
- 右侧：兑换码激活按钮
