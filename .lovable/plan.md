

## 个性化"7天后收获" -- 行为/情绪/信念三维度定制

将静态的3条通用收获替换为基于测评结果动态生成的3条个性化成长目标，分别对应行为、情绪、信念三个维度。

### 当前问题

```text
现在显示的是：
[check] 清晰看见自己的财富卡点类型
[check] 一套可持续的每日觉察习惯  
[check] AI记录的个人成长档案
```

这3条是固定文案，与用户的具体卡点和反应模式无关。

### 改造后效果（示例：嘴穷 + 追逐型用户）

```text
7天后，你会得到：
🎯 行为突破：从"嘴穷"到"嘴富"，学会用丰盛语言表达自己的价值
💭 情绪松绑：减少追逐式焦虑，建立与金钱的从容关系
💡 信念升级：植入"我值得丰盛"的新信念，替换匮乏底层程序
```

### 具体操作

**文件1**：`src/components/wealth-block/AwakeningJourneyPreview.tsx`

1. **新增 prop**：`reactionPattern?: string`
2. **删除**静态 `sevenDayOutcomes` 数组（第 57-62 行）
3. **新增**个性化收获生成逻辑：

按 `dominantPoor` 生成行为维度收获：

| 卡点类型 | 行为收获 |
|---------|---------|
| 嘴穷 | 从"嘴穷"到"嘴富"，学会用丰盛语言表达自己的价值 |
| 手穷 | 从"手穷"到"手富"，建立给予-接收的自然流动感 |
| 眼穷 | 从"眼穷"到"眼富"，打开感恩视角看见身边的富足 |
| 心穷 | 从"心穷"到"心富"，从受害者模式切换到创造者模式 |

按 `reactionPattern` 生成情绪维度收获：

| 反应模式 | 情绪收获 |
|---------|---------|
| 追逐型 | 减少追逐式焦虑，建立与金钱的从容关系 |
| 逃避型 | 重建安全感，面对财富话题不再退缩 |
| 创伤型 | 调节神经系统，财富话题不再触发身心反应 |
| 和谐型 | 巩固情绪稳定，向丰盛型状态自然进化 |

信念维度收获（结合卡点类型）：

| 卡点类型 | 信念收获 |
|---------|---------|
| 嘴穷 | 植入"我值得被看见"的新信念，替换自我贬低程序 |
| 手穷 | 植入"给予即丰盛"的新信念，替换紧握匮乏程序 |
| 眼穷 | 植入"我已拥有很多"的新信念，替换比较不足程序 |
| 心穷 | 植入"我有力量创造"的新信念，替换受害者程序 |

4. **UI 调整**：每条收获前加对应维度 emoji 标识（🎯行为 / 💭情绪 / 💡信念），让三维结构一目了然

**文件2**：`src/components/wealth-block/WealthBlockResult.tsx`

- 传入 `reactionPattern={result.reactionPattern}` 给 `AwakeningJourneyPreview`
