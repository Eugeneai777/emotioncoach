

## 优化有劲小工具页面布局

### 当前问题

1. **布局**：测评和工具分两个区块上下排列，每个区块内用 2 列网格，视觉上不够直观
2. **描述无价值**：当前 features 写的是"56题专业评估"、"90题权威量表"、"24题精准诊断"等，用户无法理解实际价值

### 优化方案

将整个区域改为**左右双栏布局**：左列放测评产品，右列放日常工具，每列内产品纵向堆叠。同时将所有 features 描述替换为对用户有实际意义的价值说明。

### 具体修改

**文件：** `src/components/Tools99Grid.tsx`

#### 2. 改为左右双栏布局

去掉按分组（测评诊断/日常工具）分区渲染的逻辑，改为一个容器内左右两列：

```
+------------------+------------------+
|   📊 测评诊断     |   🛠 日常工具     |
+------------------+------------------+
| 🎫 尝鲜会员       | 🫀 死了吗打卡     |
| ¥9.9             | ¥9.9             |
+------------------+------------------+
| 💚 情绪健康测评   | 📔 觉察日记       |
| ¥9.9             | ¥9.9             |
+------------------+------------------+
| 📋 SCL-90测评     | 🆘 情绪SOS按钮   |
| ¥9.9             | ¥9.9             |
+------------------+------------------+
| 💰 财富卡点测评   |                  |
| ¥9.9             |                  |
+------------------+------------------+
```

实现方式：
- 外层容器使用 `grid grid-cols-2 gap-2.5`
- 左列（测评）和右列（工具）各自为一个 `div`，内部使用 `space-y-2.5` 纵向堆叠
- 每列顶部有分组标题（📊 测评诊断 / 🛠 日常工具）
- 测评有 4 个产品，工具有 3 个产品，右列底部自然留白

#### 3. Hero 横幅保持不变

保留现有的"🧰 有劲小工具"横幅和标签。
