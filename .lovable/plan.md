

# 优化三层诊断模块 - 移除重复显示

## 问题分析

当前页面存在信息重复：
- 上方有三个颜色条显示"第一层 状态筛查（12题）"、"第二层 反应模式（16题）"、"第三层 行动路径（4题）"
- 下方 Accordion 又重复显示同样的层级名称

用户希望简化，不需要显示"几题"的信息。

## 优化方案

**方案：合并两处为一体**

将上方的三层简化说明删除，只保留 Accordion 折叠表格。Accordion 标题已经清晰说明每层的内容，无需重复。

优化后结构：
```
┌─────────────────────────────────────────────────────────────┐
│  🎯 三层诊断 · 层层深入                                      │
├─────────────────────────────────────────────────────────────┤
│  🧅 洋葱图动画                                              │
│  "由外向内 · 层层剥离 · 直达情绪卡点"                        │
├─────────────────────────────────────────────────────────────┤
│  ▼ 第一层：状态筛查（科学背书层）   [展开/收起]            │
│  ▼ 第二层：反应模式（卡点诊断层）   [展开/收起]            │
│  ▼ 第三层：行动路径（转化承接层）   [展开/收起]            │
└─────────────────────────────────────────────────────────────┘
```
