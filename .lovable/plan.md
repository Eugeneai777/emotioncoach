
# 《引流+运营完整SOP手册》PDF生成方案

## 目标
把"7天加企微"和"种子群长期运营"两段流程合并成一份中文PDF，覆盖从用户加微到进群后周节奏运营的完整链路。

## 交付物
- `docs/seed-group-weekly-v1.2.md`（新建，种子群周节奏表完整版）
- `/mnt/documents/引流+运营完整SOP手册.pdf`（最终交付）

## 步骤

### 1. 落地种子群v1.2 markdown
基于历史对话定稿内容整理 `docs/seed-group-weekly-v1.2.md`，包含：
- 定位与目标人群（35-55 高压男性/女性）
- 周节奏表（周一~周日动作 + 负责人 + 话术骨架）
  - 周二 共鸣卡
  - 周四 教练陪聊
  - 周六 文字夜话
  - 周日 静默
- 第1周静默观察期规则
- 转化触点（何时推¥399 / ¥3980）
- 群规与禁忌
- 数据回收口径（留存/互动/转化）

如有定稿细节缺失会在生成前再向你确认。

### 2. 合并两份文档
按以下顺序拼装一份总markdown：
```
封面 + 目录
├─ 第一部分：7天加企微 SOP v2.0（docs/seed-group-plan.md 全文）
└─ 第二部分：种子群长期运营节奏 v1.2（新文件全文）
附录：衔接关系图（D7→进群→第1周静默→长期节奏）
```

### 3. 渲染PDF
- 工具：`pandoc + xelatex`
- 中文字体：Noto Serif/Sans CJK SC（已在沙箱安装）
- 版式：A4、1英寸边距、目录页、页眉页脚、章节分页
- 保留表格、代码块、emoji

### 4. QA（强制）
`pdftoppm` 把每一页转图片逐页人工检查：
- 中文有无乱码/方框
- 表格是否溢出
- 章节是否串页
- 目录页码是否正确

如有问题修脚本重渲染，直到全部干净。

### 5. 交付
通过 `<presentation-artifact>` 输出PDF下载链接。
