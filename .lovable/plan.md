

# 企微二维码改为弹窗展示

## 评估

完全合理。当前二维码直接外露确实抢夺了打卡任务的注意力焦点，且占用大量页面空间。改为「点击触发弹窗」的模式：
- **保留转化路径**：用户仍能一步触达二维码，不增加认知负担
- **聚焦核心任务**：打卡页的视觉重心回归到每日任务本身
- **体验更精致**：弹窗居中展示二维码，配合扫码提示，更符合小程序内的交互习惯

## 变更计划

### 文件：`src/pages/CampCheckIn.tsx`

| 变更 | 说明 |
|------|------|
| 将 `WeComGuideCard` 改为可点击卡片 | 仅保留标题和副文案区域，去掉内嵌的二维码图片 |
| 点击后弹出 `Dialog` | 居中展示企微二维码 + "长按识别二维码添加助教" 提示文字 |
| 引入 `useState` 控制弹窗 | 添加 `showQR` 状态，点击卡片 `setShowQR(true)` |

卡片保持当前的绿色渐变样式，右侧加一个箭头图标暗示可点击。弹窗使用项目已有的 `Dialog` 组件。
