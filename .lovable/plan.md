

## 教练解说按钮改为底部Tab栏中间固定按钮

将当前的浮动圆形按钮改为嵌入底部Tab导航栏中间的红色按钮，固定不动，文字在按钮内部。

### 改造内容

**布局效果**：

```text
┌──────────────┬──────────────┬──────────────┐
│  开始测评     │  教练解说    │   历史记录    │
│              │  （免费）     │              │
└──────────────┴──────────────┴──────────────┘
```

按钮固定在底部导航栏中间，页面滚动时不动。

### 文件1：`src/components/wealth-block/AssessmentVoiceCoach.tsx`

1. **移除**固定浮动布局（fixed定位、圆形按钮、呼吸光晕动画）
2. **改为**普通内联按钮组件，返回一个红色渐变按钮
3. **按钮样式**：红色渐变背景，白色文字，圆角，Mic图标 + 文字在按钮内部
4. **文字逻辑**：
   - 有免费次数时：显示"教练解说（免费）"
   - 免费次数用完时：显示"教练解说"（不显示"免费"）
   - 达到上限时：显示"升级解锁"
5. **移除**"还剩 X/X 次免费体验"提示文字
6. **移除**外部motion.div包裹和呼吸动画

### 文件2：`src/pages/WealthBlockAssessment.tsx`

1. 在底部Tab导航栏的"开始测评"和"历史记录"之间插入教练解说按钮
2. 按钮只在有测评结果（`showResult` 为 true）时显示
3. 需要将 `result`、`aiInsight`、`healthScore` 等状态传递给按钮组件
4. TabsList 改为3列布局，中间列放教练解说按钮

### 技术细节

- `AssessmentVoiceCoach` 不再使用 fixed 定位，改为接收 className 或直接内联渲染
- 按钮颜色：`bg-gradient-to-r from-red-500 to-rose-500`，与两侧amber色Tab形成对比
- 按钮内容：Mic图标 + 文字，紧凑排列
- 从 `WealthBlockResult.tsx` 中移除 `AssessmentVoiceCoach` 的渲染（因为已移到页面级底部栏）
