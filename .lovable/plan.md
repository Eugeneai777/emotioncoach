
## 创建独立的教练入驻介绍页面

### 目标
创建一个新的独立页面 `/coach-recruitment`，作为教练招募的落地页，包含平台介绍、入驻优势、申请流程说明等内容，底部有明确的 CTA 按钮引导跳转到现有的申请表单页 `/become-coach`。这个链接可以作为外部分享链接使用。

### 新建文件

**`src/pages/CoachRecruitment.tsx`** — 教练招募落地页

页面结构：

1. **顶部 Hero 区域**
   - 大标题："加入有劲，成为认证教练"
   - 副标题简要说明平台定位
   - 主 CTA 按钮："立即申请入驻"

2. **平台介绍**
   - 简要介绍有劲平台是什么
   - 平台用户规模 / 愿景

3. **入驻优势（3-4 项卡片）**
   - 专业背书：平台认证，提升个人品牌
   - 精准获客：平台流量导入，匹配合适用户
   - 灵活自主：自定义服务项目和定价
   - 工具支持：在线预约、日程管理等工具

4. **申请流程说明（4 步）**
   - 填写基本信息 → 上传资质证书 → 设置服务项目 → 提交审核
   - 与现有 BecomeCoach 页面的 4 个步骤对应

5. **常见问题 FAQ（手风琴折叠）**
   - 入驻需要什么条件？
   - 审核需要多长时间？
   - 如何设置服务和收费？
   - 平台如何收费？

6. **底部 CTA**
   - 再次放置"立即申请入驻"按钮

### 修改文件

**`src/App.tsx`** — 添加路由

在现有的 `/become-coach` 路由附近添加：
```
<Route path="/coach-recruitment" element={<CoachRecruitment />} />
```

### 技术细节

- 使用现有的 UI 组件（Card, Button, Accordion 等）
- 使用 `DynamicOGMeta` 设置 OG 分享标签，便于外部链接分享时显示标题和描述
- 页面无需登录即可访问（公开页面）
- 点击申请按钮跳转到 `/become-coach`
- 移动端响应式设计，风格与现有页面保持一致
