

# 简化模块1（Hero区）文案方案

## 一、当前问题分析

模块1（紫色 Hero 区）当前文字量：
```
你有没有这种感觉？
明明没什么大事
就是「怎么都提不起劲」
不是你不够努力
是有个东西一直在暗中消耗你的能量
```

**共5行文字**，信息密度高但显得冗长。

---

## 二、简化方案

将5行精简为**核心3行**：

```
明明没什么大事
就是「怎么都提不起劲」
是什么在暗中消耗你？
```

**删除内容：**
- "你有没有这种感觉？" → 删除（开场白冗余）
- "不是你不够努力" → 删除（接纳语气已通过后文传达）
- "是有个东西一直在暗中消耗你的能量" → 简化为"是什么在暗中消耗你？"（用问句激发好奇）
