

# 四大反应模式详情增强方案

## 一、需求分析

用户希望为四大反应模式添加完整详细的描述体系，包括：

| 模块 | 内容 | 用途 |
|------|------|------|
| 戳心总结 | 一句话看懂当前状态 | 快速共鸣 |
| 典型表现 | 4个可能正在经历的场景 | 自我确认 |
| 内在机制 | 背后的真实原因 | 深度理解 |
| 当下需要 | 现在最需要做的事 | 方向引导 |
| AI陪伴 | AI教练下一步会怎么陪 | 产品预热 |
| 推荐路径 | AI教练 + 训练营 | 转化路径 |
| 统一承接 | 底部转化文案 | 强转化 |

---

## 二、当前实现状态

`patternConfig` 已包含部分字段，需要增强的内容：

| 字段 | 当前状态 | 需要增强 |
|------|----------|----------|
| name, emoji | ✅ 已有 | - |
| tagline | ✅ 已有（但简短） | 需要升级为完整戳心句 |
| description | ✅ 已有 | 与tagline合并优化 |
| symptoms | ✅ 已有 | 内容需更新为用户提供版本 |
| mechanism | ✅ 已有 | 内容需更新 |
| need | ✅ 已有（简短） | 需要拆分为数组 |
| aiOpening | ✅ 已有 | - |
| aiNextStep | ⚠️ 缺失 | 新增完整AI陪伴说明 |
| recommendedCoach/Camp | ✅ 已有 | - |
| firstStep | ✅ 已有 | - |

## 三、数据层更新

### 3.1 更新 `patternConfig` 结构

```typescript
export const patternConfig: Record<PatternType, {
  // 基础信息
  name: string;
  emoji: string;
  color: string;
  bgColor: string;
  targetAudience: string;
  
  // 戳心总结（新增独立字段）
  headline: string;  // "一句话看懂你现在的状态"
  
  // 典型表现（更新内容）
  symptoms: string[];
  
  // 内在机制（更新内容）
  mechanism: string;
  
  // 当下需要（拆分为数组）
  currentNeeds: string[];  // 现在最需要先做的
  needsContext: string;    // 不是改变，而是修复
  
  // AI陪伴说明（新增）
  aiNextStep: string;  // AI教练下一步会这样陪你
  
  // 推荐路径
  recommendedCoachLabel: string;  // "情绪修复 AI 教练"
  recommendedCampLabel: string;   // "能量恢复训练营"
  
  // 对话开场
  aiOpening: string;
  
  // 今日第一步
  firstStepTitle: string;
  firstStepDescription: string;
}>;
```
