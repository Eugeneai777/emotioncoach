
# 觉察日记入口页重新设计

## 问题分析

### 当前问题
1. **标题不清晰**：「觉察入口」概念抽象，用户不知道这是什么
2. **6个入口没有解释**：为什么是这6个？它们之间有什么关系？
3. **缺乏动机**：没有告诉用户「为什么要做这件事」
4. **介绍文案过于简单**：「每天1次轻记录」不足以打动人

### 文案设计（基于用户提供的内容）

#### Hero区域
```
✨ 频繁记录自己，可以改命
这不是玄学，是神经科学
```

#### 科学依据卡片
```
📊 研究表明
• 人类每天遗忘 70% 的经历
• 持续记录者 5年后决策速度快 3倍
• 职业转变或创业效率提升 200%
```

#### 痛点共鸣区（可折叠）
```
你是否也这样？
• 每天机械化起床、上班、刷手机
• 上周发生的事，模模糊糊没印象
• 在信息茧房里「自动驾驶」
• 感觉迷茫混乱，抓不住重点

记录 = 用最低成本打破无意识状态
```

#### 6入口分类说明

**困境记录（3个）**
- 🔥 **情绪** - 看见被忽略的信号
- 🧩 **选择** - 理清内心的冲突
- 🤝 **关系** - 表达未被听见的话

*写法提示：不叫「困难」，叫「我的破局关键点」或「命运转折点」*

**顺境记录（3个）**
- 💛 **感恩** - 重写神经回路
- ⚡ **行动** - 把碎片变成复利
- 🌟 **方向** - 点亮人生主线

*写法提示：记录微小美好，散步、电影、灵感、三餐*

#### 底部金句
```
把平凡日常积累成个人成长的复利资产
将碎片化时间冶炼成金
```

## 关键设计决策

### 为什么是6个入口？

现在将明确分为两类：

1. **困境记录**（情绪、选择、关系）
   - 当你遇到负面情绪、纠结、摩擦时
   - 写法：把困难重新命名为「破局关键点」

2. **顺境记录**（感恩、行动、方向）
   - 记录让你感到美好、有能量的时刻
   - 写法：捕捉微小美好，积累复利

这种分类让用户一目了然：**无论遇到什么，总有一个入口适合你**
