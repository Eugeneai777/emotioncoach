我会把之前 PDF 里的“短视频选题库”做成一个可持续自动生成的页面，后续你不用每次让我手动整理。

## 页面目标
新增一个内容工作台页面，例如：`/mini-app-content-lab`。

页面用于围绕 `/mini-app` 的功能自动生成短视频选题列表，每条内容包含你要的 5 个字段：

1. 痛点
2. 价值
3. 搭配测评/工具
4. 附加价值 AI 分析报告
5. 小红书爆款标题

同时增加可复制、可筛选、可导出的能力，方便你直接做一系列短视频。

## 页面功能

### 1. 自动选题生成
页面顶部提供生成配置：

- 目标人群：通用、女性专区、中年觉醒、职场解压、情侣夫妻、银发陪伴、青少年
- 内容来源：/mini-app 场景、日常工具、专业测评、训练营/产品转化
- 生成数量：10 / 20 / 30 条
- 内容风格：小红书种草、抖音口播、情绪共鸣、测评转化

点击“生成选题库”后，调用现有 Lovable AI 能力自动生成结构化列表。

### 2. 固定字段表格
生成结果展示为卡片/表格两种视图，字段固定为：

```text
痛点 | 核心价值 | 搭配测评/工具 | AI分析报告附加价值 | 小红书爆款标题
```

每条内容还会补充：

- 推荐视频开场 Hook
- 适合推广入口
- 建议 CTA，例如“评论区测一测”“私信领取入口”“去小程序做测评”

### 3. 一键复制与批量导出
每条选题支持：

- 复制单条
- 复制爆款标题
- 复制完整口播结构

整页支持：

- 导出 CSV，方便你放进表格做排期
- 导出 Markdown，方便继续整理成文档
- 后续如需要，也可以再加“导出 PDF”按钮

### 4. 与现有视频生成器衔接
现在项目已有 `/video-generator`，里面已经有 AI 剧本生成和数字人视频生成逻辑。

我会让新页面生成的选题可以：

- 点击“生成口播稿”跳转或带参数进入 `/video-generator`
- 自动带入主题、人群、转化产品
- 后续你可以从选题库直接进入视频生产流程

### 5. 内容来源自动同步
页面会复用现有配置和数据：

- `/mini-app` 当前的人群入口、场景入口、用户见证、轮播转化点
- `videoScriptConfig.ts` 里的主题、人群、转化产品
- 后台已有的 AI 教练、训练营数据

这样以后你新增工具、测评、训练营时，这个页面也能拿到最新内容，不需要每次重新写死。

## 技术实现

### 前端
新增页面：

- `src/pages/MiniAppContentLab.tsx`

并在路由中加入：

- `/mini-app-content-lab`

页面使用现有组件体系：

- Card
- Button
- Select
- Tabs
- Table
- Textarea
- Badge

保持移动端也能用，方便你在手机上直接复制标题和选题。

### 配置复用
从现有文件复用：

- `src/config/videoScriptConfig.ts`
- `/mini-app` 中的场景和人群逻辑会整理为可复用数据，避免重复维护

如果当前 `/mini-app` 里有部分内容是页面内局部常量，我会抽出到配置文件，例如：

- `src/config/miniAppContentMap.ts`

用于统一管理“场景 -> 痛点 -> 推荐测评/工具 -> 转化产品”。

### AI 后台函数
新增一个后台函数：

- `mini-app-content-ai`

输入：

```json
{
  "audience": "workplace",
  "sourceType": "assessments",
  "count": 20,
  "style": "xiaohongshu"
}
```

输出严格 JSON：

```json
{
  "items": [
    {
      "painPoint": "...",
      "value": "...",
      "matchedTool": "...",
      "aiReportValue": "...",
      "viralTitle": "...",
      "hook": "...",
      "cta": "...",
      "route": "..."
    }
  ]
}
```

AI 调用沿用项目已有 Lovable AI Gateway，不需要你额外提供 API Key。

## 交付结果
实现后你会得到：

- 一个可访问的自动选题生成页面
- 结构化 5 字段选题列表
- 一键复制标题/整条内容
- CSV/Markdown 下载
- 可衔接现有数字人视频生成器

这个页面会比 PDF 更适合长期使用：以后你要做 30 条、50 条短视频，可以直接在页面里自动生成和导出。
