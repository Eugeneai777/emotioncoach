

# /mama 页面优化 — 升级为"女性专区"

## 当前问题

1. **品牌不一致**：入口卡片已改为"女性专区"，但页面内仍显示"宝妈AI"、"懂你的温暖陪伴"、"分享给闺蜜"、footer"宝妈AI · 温暖陪伴每一天"，定位过窄
2. **缺少转化通道**：/laoge 有顶部悬浮转化条引导至训练营，/mama 完全没有。女性用户没有从诊断到 ¥399 训练营的路径
3. **场景标签偏"宝妈"**：快速场景（"又吼孩子了""孩子不听话"）全部是育儿场景，无法覆盖单身女性和已婚未育群体
4. **测评入口跳走了**：当前点击"能量测评 & 工具"跳转到 /assessment-tools 通用页，没有留在女性专区闭环内
5. **语音教练标题仍是"宝妈AI语音教练"**

## 改动方案

### 1. 品牌文案全面更新

| 位置 | 旧文案 | 新文案 |
|------|--------|--------|
| 品牌标题 | 宝妈AI | 女性专区 |
| 副标题 | 懂你的温暖陪伴 | 懂 你 的 辛 苦 与 力 量 |
| 分享按钮 | 分享给闺蜜 | 分享给好友 |
| 语音下方提示 | 不需要坚强，这里可以放下所有 💖 | 不需要坚强，这里可以做自己 💖 |
| 语音教练标题 | 宝妈AI语音教练 | 女性AI语音教练 |
| Footer | 宝妈AI · 温暖陪伴每一天 | 女性专区 · 懂你的辛苦与力量 |
| localStorage key | preferred_audience = 'mama' | 不变（保持路由兼容） |

### 2. 添加顶部转化悬浮条（对齐 /laoge 模式）

在页面顶部增加粉色系悬浮条，引导至训练营：
- 文案：`🌸 7天有劲训练营 · 找回你的能量`
- 按钮：`了解详情 →`，跳转 `/promo/synergy?source=mama`
- 样式：`sticky top-0 z-50`，粉色渐变背景，与页面整体风格一致

### 3. 扩充快速场景标签

保留部分育儿场景，增加职场女性和情感场景，覆盖更广人群：

```text
原6个（全是育儿/宝妈）→ 新8个（混合场景）：
😤 又吼孩子了（保留）
😴 好累好累（保留）  
💔 没人理解我（保留）
😰 职场好难（新增 — 覆盖职场女性）
🌙 睡不着（保留）
💪 想变更好（新增 — 成长型需求）
🥺 想哭一会儿（保留）
😣 关系好累（新增 — 覆盖婚姻/单身情感）
```

### 4. 3列功能入口调整

当前3个入口偏情绪工具，增加一个更泛化的选项：

| 位置 | 旧 | 新 |
|------|-----|-----|
| 第1个 | 😊 情绪日记 | 😊 情绪日记（保留） |
| 第2个 | 🆘 情绪SOS | 🆘 情绪SOS（保留） |
| 第3个 | 📝 感恩日记 | ✨ 能量测评（改为直接进入测评，闭环转化） |

### 5. 测评入口内联化

将"能量测评 & 工具"按钮改为跳转至带"测一测"badge 的两个测评（35+女性竞争力、情绪健康测评），与 mini-app 入口的 badge 对齐。保留跳转 /assessment-tools 但文案更精准。
