

## 优化 /partner/benefits-all 权益页面

### 改动概要

1. **移除顶部双列权益卡片**（第44-87行） - 这些卡片与下方 matrix 对比表信息重复
2. **扩展 matrix 对比表**，加入有劲全部 3 个等级（L1/L2/L3），让用户看到升级后的权益差异
3. **标注当前等级** - 在对应列高亮显示"当前等级"标签
4. **添加升级按钮** - matrix 表底部放置升级引导按钮，跳转到 `/partner/youjin-intro`

### 技术细节

**文件**: `src/pages/PartnerBenefitsUnified.tsx`（唯一修改文件）

1. **删除**第44-87行的双列权益卡片区域
2. **扩展 ResponsiveComparison**：
   - 列从 3 列扩展为 5 列：对比项 + 绽放 + 有劲L1 + 有劲L2 + 有劲L3
   - 根据用户当前 `partner.partner_type` 和 `partner.level_name` 在对应列标注"当前"
   - 新增行：专属服务、体验包数量、价格等
3. **底部升级按钮**：
   - 当用户当前有劲等级低于 L3 时，显示升级引导按钮
   - 按钮跳转 `/partner/youjin-intro`
   - bloom 合伙人（自动含 L1）也可看到 L2/L3 的升级路径

列高亮逻辑：bloom 合伙人同时高亮"绽放"列和"有劲L1"列；纯有劲合伙人高亮其当前等级列。

无数据库改动，仅修改 1 个文件。
