## 产出 2：SOP 增补手册（PDF + md）

文件名：`/mnt/documents/引流+运营完整SOP手册_v2_增补.pdf`（及同名 `.md` 源）

**章节结构**

1. **群公告模板**（周一 19:30 新人入群触发）
   - 群名：`有劲客厅 · 01`
   - 4 个固定触点：周二 21:30 共鸣卡 / 周四教练陪聊 / 周五 12:30 问题箱 / 周六 21:30 文字夜话
   - 静默承诺：周三、周日整日 0 推送
   - 禁忌：不 @ 全员、不强制自我介绍、不发 emoji 雨
2. **共鸣卡使用说明**（周二 21:30）
   - 8 张轮换表 + 每张配 1 句群内引导话术（≤30 字，不带链接）
   - 禁忌：不追问、不要赞、24h 内不私聊
3. **问题箱话术**（周五 12:30）
   - 群发模板嵌入 `https://v.wjx.cn/vm/eQe3Egn.aspx#`
   - 问卷星后台清单：关闭"获取微信信息"、确认"匿名"承诺前置
   - 数据回收：周日 15:00 按"来源/提交时间"筛选
4. **周六文字夜话 SOP**（21:30）
   - 3 档机制：≤5 人=群内文字（默认）/ 6–12 人=文字+准教练【炯谦】入场 / ≥13 人=腾讯会议语音圆桌 30min
   - 文字流详细脚本（21:30–22:30）
   - 语音圆桌详细脚本（20:30–21:30 预约 + 进场）
   - **冷场兜底三段式**：
     - 0–10min 自然等待（不打破沉默）
     - 10–15min 换钩子（教练抛 1 个具体场景题）
     - 15–25min 自曝（必须真名 + 真事 ≤80 字）
   - 红线：禁止假回应、禁止内部人冒名接话、禁止伪造互动数据
   - 触发 v1 老规则：连续 3 次冷场 → 4 周 0 互动 → 移出种子群
5. **群名续号规则**：`有劲客厅 · 01 / 02 / 03 …`，满 80 开新群，不按性别/地域拆分
