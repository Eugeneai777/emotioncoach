

# 重构"训练营能给你带来什么"→ 清晰分层交付清单

## 问题诊断

当前 outcomes 区域是 8 个 emoji 小卡片（情绪更稳定、关系更和顺…），全是"感受描述"而非"你会拿到什么"。用户看完不知道自己买的到底包含哪些具体交付物。这在课程设计中属于"价值感知缺失"——用户需要看到**具体交付物清单 + 层级关系**，而非笼统的效果承诺。

## 设计方案：三层交付金字塔

参考得到/混沌学园课程详情页的"你将获得"模块，将 8 个虚泛 outcomes 替换为**三层结构化交付清单**：

```text
┌─────────────────────────────────────┐
│  🔥 核心交付（每天都用）              │
│  ┌───────────────────────────────┐  │
│  │ AI情绪教练系统 · 24h陪伴      │  │
│  │ 7天专业冥想音频（真人录制）    │  │
│  │ 知乐胶囊 · 草本身心调理       │  │
│  └───────────────────────────────┘  │
├─────────────────────────────────────┤
│  ⭐ 专属亮点（高价值稀缺）          │
│  ┌───────────────────────────────┐  │
│  │ 1V1教练专属辅导               │  │
│  │ 海沃塔团队研讨辅导             │  │
│  └───────────────────────────────┘  │
├─────────────────────────────────────┤
│  🎁 附加权益                        │
│  │ 成长社群 · 每日课程推荐        │  │
│  │ 7天成长报告                    │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

### 视觉层级
- **核心交付**：大卡片，左图标右文字，每个卡片有具体数字/频次说明（如"7天×10-17分钟"），橙色强调边框
- **专属亮点**：中卡片，带"稀缺"角标，紫/金色调
- **附加权益**：紧凑横排小标签，灰底淡化，传递"还有更多"但不抢主视觉

### 每个交付项包含
- 具体名称（不是感受词）
- 一句话说明频次/时长/方式
- 数字化锚点（如 "7天"、"24小时"、"10-17分钟"）
