

## 排版重设计：全屏沉浸语音 + 情境化内容区

用户选择了两个方向：**语音卡片全屏沉浸** + **增加情境化内容区**。

### 整体布局变化

当前结构：语音卡片(圆角卡片) → 四入口(小图标) → 折叠更多

改为：**全屏语音英雄区(占满首屏)** → **四入口(浮在语音区底部)** → **情境化推荐卡片** → **折叠更多**

```text
┌─────────────────────────┐
│  PageHeader (透明)       │
│                         │
│   问候语                 │
│                         │
│     ◯ 语音按钮           │  ← 占满视口，像 Calm 首页
│     呼吸光晕             │
│   "随时开口，我在"        │
│                         │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐│  ← 四入口浮在底部
│  │😔 │ │📖 │ │📊 │ │🚀 ││
│  └───┘ └───┘ └───┘ └───┘│
├─────────────────────────┤
│  🌅 情境化推荐            │  ← 新增区域
│  "今日推荐" / "最近在用"   │
│  ┌─────────┐┌─────────┐ │
│  │ 卡片1    ││ 卡片2    │ │
│  └─────────┘└─────────┘ │
├─────────────────────────┤
│  还想探索更多？ ▼         │
│  (QuickNavFooter)       │
└─────────────────────────┘
```

### 具体改动

#### 1. `SuperEntry.tsx` — 全屏沉浸重构
- 语音区从圆角卡片改为**占满视口高度**的全屏容器（`min-h-[70vh]`），去掉圆角和边框
- 背景改为全屏渐变 + 多层模糊光斑（比卡片更大更柔和）
- 问候语放大，语音按钮居中偏上
- 四入口移到语音区底部，用半透明毛玻璃底栏 `bg-white/5 backdrop-blur-md rounded-2xl` 包裹
- 整体像 Calm/小睡眠 的首页感觉

#### 2. 新增 `ContextCards.tsx` — 情境化内容区
- 在 SuperEntry 下方新增一个区域
- 显示两种内容（根据用户状态）：
  - **已登录**：显示「最近使用」的工具（从 `user_tool_usage` 查询最近3个），以横向小卡片展示
  - **未登录/无历史**：显示「今日推荐」静态卡片（如"3分钟呼吸练习"、"写一段感恩日记"）
- 卡片样式：暖色毛玻璃，圆角大，带渐变图标
- 点击卡片触发 `onInlineTool` 或导航

#### 3. `LivingLab.tsx` — 集成情境区
- 在 SuperEntry 和 Collapsible 之间插入 ContextCards
- 调整间距适配全屏语音区

### 改动文件
- `src/components/living-lab/SuperEntry.tsx` — 全屏沉浸重构
- `src/components/living-lab/ContextCards.tsx` — 新建，情境化推荐卡片
- `src/pages/LivingLab.tsx` — 集成新组件
