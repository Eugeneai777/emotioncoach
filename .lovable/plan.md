

# 底部导航栏整体视觉风格升级

## 目标
全面优化底部导航栏的视觉品质，提升高级感和一致性，让 AI教练解说按钮更突出、两侧导航更精致。

## 修改文件
1. `src/components/wealth-block/AssessmentVoiceCoach.tsx` - 按钮与价值标签视觉升级
2. `src/pages/WealthBlockAssessment.tsx` - 底部导航栏整体样式升级

## 设计方案

### 1. 底栏背景升级
- 背景改为毛玻璃效果：`backdrop-blur-xl bg-white/80`
- 边框改为更细腻的分割线：`border-t border-gray-100`
- 阴影改为更柔和的上投影

### 2. AI教练解说按钮视觉升级
- 按钮尺寸缩小至 `w-20 h-20`（从 24 缩至 20），减少对两侧的压迫
- 渐变色调整为更有质感的 `from-rose-500 via-red-500 to-rose-600`
- 脉冲光环改为双层呼吸光效，更克制不刺眼
- 图标和文字微调间距，整体更紧凑
- "免费"标签改为更精致的样式

### 3. 价值标签优化
- 改为半透明玻璃质感：`bg-white/60 backdrop-blur-sm border border-rose-100`
- 图标颜色加深，文字加粗，提升可读性
- 动画改为从按钮中心扩散出来的效果

### 4. 两侧导航按钮升级
- 图标改用更细的描边风格，选中态加粗
- 选中态加底部指示条（小圆点或短横线）
- 文字间距优化，整体更对齐
- 历史记录角标改为更小巧的圆点样式

## 技术细节
- 修改两个文件
- 使用 Tailwind CSS + framer-motion
- 不改变组件接口和功能逻辑
- 不新增依赖
