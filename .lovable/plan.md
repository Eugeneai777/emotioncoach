
# 重命名"组合产品"并优化卡片设计

## 变更概述

### 1. 重命名 + 调整 Tab 顺序

**文件：`src/components/admin/IndustryPartnerManagement.tsx`**

Tab 顺序从：`飞轮分析 | 商城商品 | 商城订单 | 产品组合包`

改为：`飞轮分析 | 组合产品 | 商城商品 | 商城订单`

### 2. 全局文案替换

**文件：`src/components/admin/industry-partners/PartnerProductBundles.tsx`**

- 标题 "产品组合包" → "组合产品"
- 空态文案 "暂无产品组合包" → "暂无组合产品"
- 对话框标题 "创建产品组合包" → "创建组合产品"
- AI 建议名称的默认值 "产品组合包" → "组合产品"

### 3. 优化卡片设计

当前卡片只是普通的 Card + 纯文字，信息密度低且缺乏视觉层次。优化方案：

**视觉提升：**
- 左侧添加主题色竖条（`border-l-4 border-primary`），与训练营卡片风格一致
- 已上架的卡片使用绿色竖条（`border-l-4 border-green-500`）区分状态
- 产品数量和价格用独立的小标签展示，而非纯文本
- AI 文案预览区加背景色（`bg-muted/50 rounded-lg p-2`）提升层次
- 操作按钮区底部分割线分隔，更清晰

**信息结构优化：**

```text
┌─────────────────────────────────────┐
│█                                    │
│█  知乐 35+ 身心觉醒包    ✅已上架    │
│█  ┌──────┐ ┌────────┐              │
│█  │3 个产品│ │¥697.90 │              │
│█  └──────┘ └────────┘              │
│█  ┌─────────────────────────────┐  │
│█  │ 目标人群：身处事业上升期...    │  │
│█  │ 痛点：深夜焦虑失眠...        │  │
│█  └─────────────────────────────┘  │
│█  ─────────────────────────────────│
│█  [编辑]  [下架商品]      [删除]   │
└─────────────────────────────────────┘
```
