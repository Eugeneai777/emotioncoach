

## 优化"探索更多"展开区为引导式叙事布局

### 问题
当前展开区内4个功能卡片是扁平的按钮式设计（左色条+一行描述+箭头），缺乏引导感和层次感。参照附件截图（类似 PathDetailCards + UseCasesSection 的风格），需要改为更有叙事感的引导式设计。

### 方案（仅改 `src/pages/MiniAppEntry.tsx`）

**1. 功能路径区 → 2×2 网格引导卡片**
- 将4个 `exploreBlocks` 从纵向列表改为 **2列网格**布局（与截图中"四条路径"一致）
- 每张卡片：顶部图标+标题+副标题，下方2-3行描述文字
- 添加板块标题："四大板块，助你持续成长"
- 卡片背景使用半透明色块 + ring 边框，点击整张卡片跳转

**2. 使用场景区 → 新增叙事引导模块**
- 在功能路径下方新增"什么时候可以找有劲AI？"场景引导区（与截图风格一致）
- 4个场景卡片纵向排列：深夜焦虑时、职场迷茫时、关系困扰时、想要成长时
- 每个场景带图标 + 标题 + 一段共情描述
- 场景卡片不可点击，纯引导作用，营造"任何时刻都在"的氛围

**3. 用户见证横滑轮播保持不变**

**4. 健康商城卡片保留在网格中**（第5个卡片单独一行居中或满宽）

### 技术细节

```text
展开区结构：
┌─────────────────────────────┐
│ 四大板块，助你持续成长        │  ← 板块标题
├──────────┬──────────────────┤
│ 🛠 日常工具 │ 📊 专业测评     │  ← 2×2 网格
│  情绪SOS…  │  PHQ-9…        │

│ 🏕 训练营  │ 🏪 健康商城     │
│  AI+真人…  │  知乐胶囊…      │
├──────────┴──────────────────┤
│ 什么时候可以找有劲AI？        │  ← 场景标题
│ 任何时刻，任何情绪，它都在     │
├─────────────────────────────┤
│ 🌙 深夜焦虑时               │  ← 纵向场景卡
│ 💼 职场迷茫时               │
│ 💗 关系困扰时               │
│ 📈 想要成长时               │

│ 收起 ↑                      │

```

- `exploreBlocks` 渲染改为 `grid grid-cols-2 gap-2`，卡片内部改为纵向布局（图标在上，标题居中，描述在下）
- 新增 `useCases` 数组（Moon/Briefcase/Heart/TrendingUp 图标），样式参照 `UseCasesSection.tsx` 的半透明色块+ring边框
- 保持 `reduceMotion` 动画适配逻辑
