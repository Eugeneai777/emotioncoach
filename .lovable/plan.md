

# 凸显"训练营 + 知乐"协同效果：1+1 > 2

## 核心问题
当前页面把训练营和知乐胶囊当作两个独立产品展示，用户感觉"有没有知乐都行"。需要强调**两者配合使用**才能全天候抵御压力。

## 改动方案

### 1. 新增"一天怎么用"场景区块（核心改动）
在产品卡片下方，用时间轴展示一天的使用场景：

```text
🌅 早上 7:30  → 10分钟训练营（调整心态出门）
💊 早餐后     → 知乐胶囊 1粒（激活身体防护）
💊 午餐后     → 知乐胶囊 1粒（对抗午后焦虑）
💊 晚餐后     → 知乐胶囊 1粒（安稳入睡）
```
配合文案："训练营练心，知乐护体，**24小时全覆盖**"

### 2. 新增"效果对比"区块：单用 vs 组合
用三列对比，视觉冲击力强：

| | 只练训练营 | 只吃知乐 | 训练营 + 知乐 |
|---|---|---|---|
| 压力缓解 | 60% | 55% | **92%** |
| 起效速度 | 7天 | 30分钟 | **即刻+持续** |
| 睡眠改善 | 中等 | 中等 | **显著** |

突出组合列，让用户直观感受"缺一不可"。

### 3. 改造"超值组合"文案
从"买训练营送胶囊"改为强调协同：
> "训练营练心智 × 知乐护身体 = 全天候抗压力"
> "单独使用效果有限，组合使用效果翻倍"

### 4. 产品卡片标注角色
- 训练营标签改为："🧠 练心智"
- 知乐胶囊标签改为："💊 护身体"
- 让用户理解各自不可替代的角色

### 5. 加入产品实拍图
将用户上传的知乐胶囊产品图复制到项目中，在胶囊卡片区域展示，增加真实感和信任度。

## 实现
- 修改 `src/pages/PromoPage.tsx`：新增"每日使用场景"和"效果对比"两个 section，改造超值组合文案和产品标签
- 复制产品图到 `src/assets/zhile-capsules.jpeg` 并在胶囊卡片中引用
- 不涉及数据库改动（新增内容硬编码在页面，因为是该推广页专属）
