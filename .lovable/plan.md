
# 种子群最小可跑节奏表 v1.1

> v1.1 更新：男性案例库存 = 0 的现实下，**周四从"案例发布"改为"教练在群陪聊 30min"**；周日圆桌不变。

## 一、定位与边界

- **定位**：未付费长尾池 · 长期养鱼（非转化群、非日更群）
- **入群门槛**：D+7 仍未购买的用户才进群（D0–D7 仅私信）
- **运营人手**：1 人 + 教练每周 ≤1h
- **群规模**：单群 ≤80 人，男女合并 1 个匿名群（满 80 再开第 2 群）

---

## 二、每周节奏表（3 个固定触点）

```text
            周一  周二      周三  周四          周五  周六  周日
群内动作    静默  10:00卡   静默  20:30教练       静默  静默  21:00圆桌
                +问题箱          陪聊30min                     (60min)
运营工时    0     0.5h      0    0.5h(主持)     0    0    1.5h
教练工时    0     0         0    0.5h           0    0    0
```

### 周二 10:00 ·「一张卡 + 匿名问题箱」（运营 0.5h）
- 1 张痛点共鸣图卡（≤60 字），附匿名问题箱 H5
- 不 @所有人

### 周四 20:30 ·「教练在群陪聊 30min」（教练 0.5h + 运营 0.5h 主持）—— v1.1 新形态
**取代原案例发布**。零案例库存下，用"教练真人在场"替代"脱敏案例"。
- 教练（炯谦或其他）进群挂 30min，文字 + 1–2 条短语音
- 运营提前 1 天在问题箱里挑 3 个匿名问题作为暖场
- 教练**只做轻回应**：共鸣、把话题打开、不做诊断、不发方案
- 男女议题轮排（单周男向：事业失速/角色压力/性与亲密的"侧面切入"；双周女向：关系内耗/自我价值/身体信号）
- 结束后运营发 1 条总结海报，沉淀到群公告

> 副产品：**这 30min 就是男性案例的生产线**。教练 + 用户的真实问答 → 运营脱敏改写 → 4–6 周后即可形成首批男性案例库，反哺未来的"案例发布"形态。

### 周日 21:00 ·「匿名圆桌」（运营 1.5h）
- 60min 语音房，≤10 人，不录音
- 男女分场轮排（单周男场 / 双周女场），合并群、轮排时间，建群次数=0
- 运营主持："听 + 传"，不诊断
- 不推 ¥399；软入口留到次周二卡片底部

## 三、男性案例零库存的过渡方案

| 阶段       | 周次      | 周四形态                         | 案例库状态           |
|----------|---------|------------------------------|-----------------|
| 起跑期    | W1–W4   | 教练陪聊 30min                  | 0 → 沉淀中         |
| 萌芽期    | W5–W8   | 教练陪聊 25min + 1 条脱敏小故事 5min | 出现 2–4 条男性脱敏微案例 |
| 成熟期    | W9+     | 教练陪聊 15min + 案例发布 15min   | 进入"案例 + 陪聊"双轨    |

女性轨同步进行，但因有大课素材，W1 起即可在教练陪聊中穿插 1 条女性脱敏短案例。

## 四、内容储备表（v1.1 调整）

- **男性**：不再硬凑 12 周案例。W1–W4 只准备「陪聊议题卡」4 张（事业失速 / 父职压力 / 兄弟决裂 / 中年身体），每周 1 张作教练开场
- **女性**：W1 起每周 1 条脱敏故事（大课已有素材库可支撑 8–10 周）
- backlog 飞书表 4 列：编号 / 主题 / 来源 / 发布周次；低于 4 条预警

---

## 五、教练介入（最小化）

- 每周 1 次：**周四 20:30 群内陪聊 30min**（替代原"5min 语音统答"）
- 不做 1v1、不进私聊
- 陪聊素材由运营提前 24h 喂 3 个匿名问题

---

## 六、转化目标与北极星

- **唯一北极星**：D+7 入群 → 第 30 天仍在群 且 ≥1 次互动（圆桌/陪聊/问题箱）的人数
- **次级**：周二卡片打开率、周四陪聊在线人数 + 发言人数、周日圆桌到场率
- **¥399 软入口**：仅周二卡片底部一行 + 圆桌主持结束前 1 句口播，不发链接轰炸

## 七、运营总工时

| 动作               | 工时/周 |
|------------------|------|
| 周二卡 + 问题箱         | 0.5h |
| 周四教练陪聊主持 + 问题预筛   | 1h   |
| 周日圆桌主持            | 1.5h |
| 问题箱周汇总 + 陪聊脱敏整理   | 0.5h |
| backlog 维护         | 0.5h |
| **运营合计**          | **4h** |
| 教练（周四陪聊）          | 0.5h |

1 人运营 4h/周 + 教练 0.5h/周即可独立支撑单群 80 人。

## 八、与 D0–D7 SOP 边界

- D0–D7：仅私信，不进群
- D+7 未购买 → "种子群邀请话术" → 进群
- 进群后 30 天内不再私信营销
- 第 30 天无互动 → 转月度脉冲池（每月 1 条问候）

## 九、90 天退出仪式

- 每 90 天对"0 互动 + 0 打开"用户发 1 条退群确认私信
- 24h 无回复 → 静默移出
