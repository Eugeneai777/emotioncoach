

# 优化训练营卡片设计

## 概述
对 `TrainingCampCard` 进行视觉和布局优化，使其更紧凑、精致，关键信息更突出。

---

## 具体改动

### 修改文件
| 文件 | 说明 |
|------|------|
| `src/components/camp/TrainingCampCard.tsx` | 重新设计卡片布局与视觉 |

### 布局优化 - 更紧凑

- 减小整体 padding：`p-5` 改为 `p-4`
- 缩小各区块之间的间距：`mb-4` 统一缩减为 `mb-3`
- 合并 Header 与状态行为单行：将"财富觉醒训练营"标题、"第3天 · 待打卡"和"连续1天"徽章合并到同一行，减少纵向空间占用

### 视觉优化 - 更精致

- 卡片增加更明显的圆角：`rounded-2xl`
- 添加微妙的内阴影增加层次感：`shadow-[inset_0_1px_0_rgba(255,255,255,0.5)]`
- 里程碑图标改用更精致的圆形小指示器，已完成的带发光效果
- 进度条改为渐变色填充，而非纯色，与主题色一致
- 按钮区域减小高度（h-10），使用更圆润的圆角

### 信息突出 - 更醒目

- 当前天数和完成天数使用更大更粗的字体（text-lg font-bold）
- "待打卡"状态使用橙色背景小标签而非纯文字
- 进度百分比使用主题强调色加粗显示
- 毕业目标行增加图标着色和背景色块突出

### 改造后的视觉结构

```text
+--------------------------------------------------+
| 🏕️ 财富觉醒训练营  第3天 · [待打卡]   🔥连续1天   |
|                                                    |
|  🌱启程 ----[====]---- ⭐中期 -------- 🏆毕业     |
|       已完成 1/7 天 (14%)                          |
|                                                    |
|  🎯 毕业目标：觉醒分达到 85 分                       |
|                                                    |
|  [===== 立即打卡 =====]  [== 介绍 ==]              |
+--------------------------------------------------+
```

### 技术要点

- 所有改动限制在 `TrainingCampCard.tsx` 单个文件内
- 保持现有的主题颜色系统（`getThemeColors`）不变
- 保持所有现有交互逻辑（导航、打卡判断等）
- 仅调整 Tailwind 类名和 JSX 结构
