# 《7天加微SOP v2.0》MVP节奏表

> 目标用户：中年男/女，加企微后7天内完成"筛选 → 体验 → 进种子群"漏斗。
> 运营规模：**1人运营 + 1教练（轻介入）**，日处理上限 ≈ 30 加微/天。
> 衔接：D+7 未购用户 → 进种子群（按 v1.2 周节奏表运营）。

---

## 一、漏斗目标（先把分母定死）

| 阶段 | 节点 | MVP 目标转化率 | 说明 |
|---|---|---|---|
| 加微 | D0 | — | 分母 |
| 完成测评 | D0–D1 | ≥ 35% | 唯一硬动作 |
| D2–D5 触达回复 ≥1次 | D5 末 | ≥ 25% | 行为信号 |
| 体验/付费 ¥399 | D+3 起 | ≥ 5% | 不在D0推 |
| 进种子群（未付费） | D7 | ≥ 15%（占加微数） | 门槛≥20人才开群 |

> 凡 D2–D5 累计回复 **≥ 2 次** 的用户 = "公开课提醒组"（见 P1-⑥ 举例）。

## 三、P0 关键 SOP 详解

### ① D0 极简化（修复"流程过重"）

**只做 1 件事**：人味招呼 + 1 个开放问题。
```
模板骨架（运营必填【】内容）：
"哥/姐你好，我是XX，看到你是从【渠道名】过来的。
冒昧问一句：最近【一句你最在意的事】，是因为什么开始关注我们的？"
```
- ❌ 不发：测评链接、海报、emoji轰炸、产品介绍
- ✅ 只挂 1 个企微"欢迎语"自动文 + 5分钟内人工补一句（区分自动/真人）

### ② 付费节点后移到 D+3（修复"D0推399自杀"）

- D0–D2 **禁止** 出现 ¥399、付费、购买、报名 任何字眼
- D3 起，仅对"完成测评 + D1–D2 回复≥1次"的用户，在 D4 语音中以"体验装"口吻提及
- D6 仅对"公开课提醒组"二次提及

### ③ 模板"骨架+原话关键词+3套语序" SOP（MVP 1人版）

**核心思路**：模板不是复制粘贴文案，是 **填空题 + 抽签**。

**3步操作法**（运营每条消息发送前 ≤ 30秒）：

1. **找原话关键词**：从用户最近1条回复里，**复制1个名词或动词**（如"加班"/"孩子高三"/"体检"/"睡不好"）。
2. **选骨架**：按用户类型（枯竭男/焦虑男/枯竭女/焦虑女）打开对应骨架卡（共4张，飞书/Notion存）。
3. **抽语序**：骨架内部预置 **3套语序变体**（A/B/C），按当天日期 mod 3 决定用哪套。

**骨架卡示例（D2 枯竭男 语音文案）**：
```
关键词槽位：【用户原话】
变体A（共情前置）：
  "听你说【用户原话】这事...我之前带过一个哥，跟你状态特别像。
   今天不聊产品，就发你一段3分钟的话，听完不用回我。"
变体B（动作前置）：
  "哥，发你段语音。关于【用户原话】这个事，我想说3句话。
   听完你要是想接一句就接，不想接就放着。"
变体C（提问前置）：
  "想问你一句——【用户原话】这种状态，多久了？
   不用现在回，先听段语音。"
```

**MVP 落地工具**：
- 飞书多维表格 1 张：4类 × 7天 = 28 个骨架卡（D2/D4 含语音脚本）
- 运营企微侧边栏挂"快捷回复"，每条骨架 3 个变体并列
- **日期 mod 3 规则**：当天日期 1/4/7/10/...→ A；2/5/8/...→ B；3/6/9/...→ C

> ⚠️ 单运营 MVP 阶段，**不做用户级A/B轮换，只做日期轮换**，最简单防截图比对。

### ④ 不回复路径 3 段式触达 SOP

**触发**：用户在某日 D 未回复任何消息。

| 第几次 | 时间点 | 动作 | 内容方向 |
|---|---|---|---|
| 1 | 当日 21:30 | 文字短句 | "今天没打扰到你吧？发完就走，晚安。" |
| 2 | 次日 12:30 | 1张共鸣卡（图） | 不带问题，不带链接 |
| 3 | 第3日 7:30 | 一句关心 | "前两天发的你看到就好，不用回。" |

**3次仍 0 回复** → 打 `冷标签`，进 **月度唤醒池**（每月1号统一发1条），**退出7天SOP**。

**关键词主动退出**（用户回"别"/"不要"/"删"/"打扰"/"取消"任一）→ 立刻打 `免打扰` 标签，30天后唤醒1次，仍无回复 → 永久静默。

## 四、P1 重要修正

### ⑤ Day2–Day5 语音产能补充说明

> 用户反馈：D2–D5 差异化语音原意是"完成测评后发预录语音邀请进群"，**未考虑未进群后续触达**。

**v2.0 修正**：
- D2 / D4 **才发语音**（每天 1 条，共 2 条），其余 D3 / D5 改为文字+表情+卡片（降产能）
- 语音 = **预录音库**，4类 × 2天 = **8条**（不是16条），一次性录完
- D2 语音 = "邀请听一段"（不带任何 CTA）
- D4 语音 = "体验装介绍"（含¥399，仅对回复过的用户发）
- **未进群 ≠ 未触达**：D2 发完语音后，无论是否进群，都继续按 D3/D4/D5 节奏走，**进群动作只在 D7 统一做**

### ⑥ 行为信号"回复≥2次"举例说明

**定义**：D1–D5 共 5 天内，用户回复消息（文字/语音/表情包均算1次）**累计 ≥ 2 次**。

**举例**：
- 用户 A：D1 回"好的"，D3 回"嗯"  → **2次 ✅ 进公开课组**
- 用户 B：D1 回"好的好的好的"（连发3条）→ 算 1 次（同日合并）→ ❌ 不进
- 用户 C：D2 发了语音表情，D4 回"在看"→ **2次 ✅ 进**
- 用户 D：5天全静默 → ❌ 进"轻触达组"，D6 只发1张图，不发链接

**操作**：运营在企微侧边栏给每个用户打 `回复+1` 标签，D5 晚上数标签数分组。

### ⑦ D7 进群门槛 ≥ 20 人

- D7 19:00 统计"当周公开课组 + 完成测评未购用户"人数
- ≥ 20 人 → 21:30 发邀请话术
- < 20 人 → 顺延到下周日（用户继续待在私信池）

**邀请话术（具体事件钩子）**：
```
"这周凑齐了一批跟你状态接近的人，明早我开个小群，
不卖课、不打卡，就一个事——【本周热议事件，如"国庆前夕的焦虑"】。
你要不要进来听听？回'1'我拉你。"
```

### ⑧ 主动退出机制
已合并到 P0-④ 关键词触发，不重复。

## 五、P2 边界压测修正

- **⑨ 高峰并发**：企微"欢迎语"自动应答 + 人工 5 分钟内补一句真人话，单日上限 30人，超出排次日
- **⑩ 时间锚点**：统一 = 加微次日为 D+1，所有动作锁在 7:30 / 12:30 / 21:30 三时段池
- **⑫ 男性案例库为0**：D6 男性侧 **不发学员案例**，改发"教练侧剖析"（教练录制 1 条 2min 通用语音，4–6周后种子群产出脱敏故事再替换）

## 六、MVP 周工时核算（1运营 + 1教练）

| 角色 | 项目 | 时长 |
|---|---|---|
| 运营 | 30人/天 × 平均0.13h = 4h/天 × 7 | ~28h/week |
| 运营 | 骨架卡维护 + 标签整理 | 2h/week |
| 教练 | 预录音库（一次性，可复用4周） | 首周3h，后续0.5h/week |
| 教练 | D6 男性"教练剖析"语音 | 0.25h/week |
| **合计** | | **运营 ~30h/week，教练 ~0.75h/week** |

> ⚠️ 已逼近 1 人运营上限。**若加微>30/天，必须降日均接待量或加人**，不要靠延长时间硬扛。
