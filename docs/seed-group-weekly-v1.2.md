# 《种子群最小节奏表 v1.2》长期运营SOP

> 适用对象：完成《7天加微SOP v2.0》D7邀请、达到 ≥20 人门槛后进群的种子用户。
> 运营规模：**1运营 + 1教练（每周轻介入 ≤2 次）**，群规模上限 50 人。
> 核心目的：低密度陪伴 → 行为信号沉淀 → 自然转化 ¥399 / ¥3980，**不做硬推**。

---

## 一、定位与目标人群

- **画像**：35–55 岁高压男女（职场中层、二胎家庭、慢病/亚健康、情绪枯竭/焦虑）。
- **入群预期**：不是学习群、不是打卡群、不是带货群，是**"有人懂、不用回"的安静角落**。
- **运营哲学**：宁可冷场，不要尬聊；宁可少发，不要硬推。

---

```text
星期  动作密度  负责人    核心动作                      时段
周一  ☆        运营      新成员入群 + 群规复述(置顶)    19:30
周二  ★        运营      共鸣卡 1 张(图文)              21:30

周日  ☆        —         静默(运营/教练完全不出现)        —
```

**周触达密度**：4 次主动动作（周二/四/五/六），3 天静默（周一仅置顶、周三/日不出现）。

## 三、第1周静默观察期（关键）

新成员入群后的**第1个完整自然周**：
- 运营**不 @ 任何新人**、不发欢迎仪式、不要求自我介绍
- 周一仅置顶一条群规（见第六节）
- 周二/四/五/六照常按节奏发，但**不点名新人**
- 教练周四陪聊**只回应已发言老人**，不主动拉新人

**目的**：让新人先观察"这群是不是安全"，第2周再自然融入。

---

## 四、四个核心动作 SOP

### ① 周二 共鸣卡（运营，21:30）

- **形式**：1 张图 + ≤30 字文案，无链接、无 CTA、无问题。
- **主题池**（4 周轮换）：
  1. 中年体检报告焦虑
  2. 父母住院的那个电话
  3. 周日晚上的胸口发闷
  4. 孩子高三/中考前的失眠
- **发完动作**：发完即离开，**不催回复、不追问**。
- **行为信号**：谁点赞/谁回"+1"/谁发表情 → 运营侧边栏打 `共鸣+1` 标签。

### ② 周四 教练陪聊（教练，21:00–21:30，共 30min）

- **形式**：教练真人出现，纯文字或 1–2 条语音回应群内已有话题。
- **铁律**：
  - ❌ 不主动开话题、不发产品、不发链接
  - ❌ 不 @ 任何人
  - ✅ 只回应"当周已经发过言的人"
  - ✅ 30min 到点离开，留一句"先下了，明天见"
- **如果群内全程无人发言**：教练发 1 句"今晚就陪到这，大家安"，**不强行制造话题**。

### ③ 周五 问题箱（运营，12:30）

- **形式**：发问卷星匿名链接 1 条 + 1 句话引导。
- **话术**：
  ```
  "本周问题箱开了，匿名的。
  哪怕只想骂两句、说两句没人听的话，都可以。
  我们周日下午看，下周二的共鸣卡会从里面挑一条做。"
  ```
- **闭环**：周日运营+教练私下看问题箱，挑 1 条做下周二共鸣卡素材（**显性闭环**是种子群活下来的关键）。

### ④ 周六 文字夜话（运营，21:30）

- **形式**：开放话题 1 个，**纯文字**，运营抛完即退。
- **话题骨架**（4 周轮换，按当周序数 mod 4）：
  1. "这周最想躲开的一件事是什么？"
  2. "上次睡好觉是什么时候？"
  3. "如果今晚能不回任何消息，你想干嘛？"
  4. "最近一次觉得'还行'，是因为什么？"
- **互动规则**：
  - 用户回 → 运营**只发表情/"嗯"/"懂"**，不展开
  - 用户互回 → 运营**完全不插话**
  - 22:30 后无新消息 → 不补刀

## 五、转化触点（什么时候、对谁、推什么）

**总原则：不在群内推，只在私信推。**

| 信号 | 私信推什么 | 谁来推 | 时机 |
|---|---|---|---|
| 共鸣卡连续 2 周点赞 | ¥9.9 测评（如未做） | 运营 | 周三 12:30 |
| 周四陪聊有过 ≥1 次发言 | ¥399 体验装（7天营） | 运营 | 周五 21:30 |
| 已购¥399 + 周四陪聊 ≥2 次主动发言 | ¥3980 私教/Bloom | 教练亲自 | 完课后第3天 |
| 问题箱投稿 + 内容深度高 | 教练 1v1 文字陪聊 15min（免费） | 教练 | 周一 21:30 |

**禁忌**：

- 群内任何形式的产品截图 / 价格 / 链接
- "限时" / "倒计时" / "仅剩X名" 等促销话术
- 教练私信带任何价格数字（让运营来报价）

## 七、数据回收口径（运营每周日 15:00 填表）

| 指标 | 口径 | MVP 目标（第4周起） |
|---|---|---|
| 周活跃率 | 当周发过 ≥1 条消息的人数 / 群总人数 | ≥ 30% |
| 共鸣卡互动率 | 周二共鸣卡点赞+回复数 / 群总人数 | ≥ 20% |
| 周四陪聊参与人数 | 21:00–21:30 内发过言的不重复人数 | ≥ 5 人 |
| 问题箱投稿数 | 周五–周日累计 | ≥ 3 条 |
| ¥399 周转化 | 当周私信¥399 → 当周完成支付 | ≥ 2 单 |
| 退群人数 | 当周主动退群 | ≤ 2 人 |

**红线**：周活跃率连续 2 周 < 15% → 触发"群健康复盘"，可能合群或换主题。
